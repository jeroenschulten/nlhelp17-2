Dialoogvenster Gebruiker Eigenschappen

U gebruikt het Gebruiker Eigenschappen dialoogvenster om uw gebruikersnaam bij uw eigen gebruikersgegevens of de gebruikersnaam van de op dit moment geselecteerde gebruiker te bewerken, om de gebruiker aan de passende security groep toe te wijzen - welke hun toegangsrechten definieert - en om de view te specificeren die standaard moet verschijnen wanneer de gebruiker het project opent. U kunt dit dialoogvenster ook gebruiken om het wachtwoord van de gebruiker te veranderen.

Hoe toegang te krijgen tot het Gebruiker Eigenschappen dialoogvenster

Om toegang te krijgen tot het Gebruiker Eigenschappen dialoogvenster, rechtsklik op een gebruiker in Bibliotheek Verkenner en selecteer Eigenschappen.

 Verklaring van de velden en knoppen op het dialoogvenster
Veld Omschrijving
Naam Voer de naam van de gebruiker in.
Security groep

Selecteer de security groep waarbij de gebruiker zou moeten horen. De security groep waartoe een gebruiker hoort bepaalt de toegangsrechten van de gebruiker bij het project.

 

Klik hier voor meer informatie.
Eigenschappen Klik op deze knop om de eigenschappen van de geselecteerde security groep te bewerken.
Standaard view

Selecteer een view die standaard moet gebruikt worden wanneer de gebruiker het project opent, mits Gebruiker standaard view geselecteerd staat in het Herladen bij openen project veld op het View tabblad van het Opties dialoogvenster.

 

Klik hier voor meer informatie.
Verander Wachtwoord

Klik op deze knop om het Wachtwoorden dialoogvenster weer te geven, dat u kunt gebruiken om het wachtwoord van de gebruiker te veranderen.

 

Klik hier voor meer informatie.
... de naam van een gebruiker wijzigen?

Voer een nieuwe naam van de gebruiker in bij het Naam veld.

 ... de gebruiker toewijzen aan een security groep welke de toegangsrechten bepaalt?

Selecteer de security groep met de juiste toegangsrechten in het Security groep veld.

 ... de standaard te gebruiken view specificeren wanneer een gebruiker het project opent?

Selecteer een view die standaard moet gebruikt worden wanneer de gebruiker het project opent - mits Gebruiker standaard view geselecteerd staat in het Herladen bij openen project veld op het View tabblad van het Opties dialoogvenster - in het Standaard view veld.

 ... het wachtwoord van de gebruiker veranderen.?

Klik op Verander Wachtwoord om het Wachtwoorden dialoogvenster weer te geven, dat u kunt gebruiken om het wachtwoord van de gebruiker te veranderen.
