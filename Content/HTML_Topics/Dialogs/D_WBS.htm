Dialoogvenster Work Breakdown Structure

U gebruikt het Work Breakdown Structure dialoogvenster om een work breakdown structure op te zetten en te onderhouden waarmee u de taken van een project kunt definiëren en groeperen. De work breakdown structure, hetgeen een hiërarchische boomstructuur van WBS codes is, wordt weergegeven in het hoofdpaneel van het dialoogvenster.

Hoe toegang te krijgen tot het Work Breakdown Structure dialoogvenster

Om toegang te krijgen tot het Work Breakdown Structure dialoogvenster, op het Project tabblad, in de Eigenschappen groep, klikt u op Work Breakdown Structure.

 Verklaring van de velden en knoppen in het dialoogvenster

Het dialoogvenster toont de volgende informatie over elke WBS code in de hiërarchie:

Veld Omschrijving
WBS code Een alfanumeriek id-nummer, dat u kunt bewerken. Nieuwe projecten zullen een initiële WBS code krijgen, overgenomen van de korte naam van het project. Het dialoogvenster toont het volledige hiërarchische pad bij elke WBS code; wanneer u een WBS code wijzigt, wordt alleen de code zelf - niet het pad - om te wijzigen weergegeven.
WBS naam Een beschrijvende titel. Voor de initiële WBS code in nieuwe projecten, dit zal worden overgenomen van de lange naam van het project.
Taak aantal Het aantal taken waaraan de WBS code is toegewezen.
Startdatum De startdatum van de eerste taak waaraan de WBS code is toegewezen.
Einddatum De einddatum van de laatste taak waaraan de WBS code is toegewezen.

Het dialoogvenster toont de volgende knoppen:

Knop Omschrijving
Toevoegen Selecteer een WBS code, klik dan op deze knop om een WBS code toe te voegen aan de work breakdown structure als een 'kind' van de geselecteerde code. Bijvoorbeeld, wanneer u een nieuwe WBS code wilt creëren direct onder de hoogste-niveau code, selecteert u de hoogste-niveau code en klikt u op deze knop.
Verwijderen Klik op deze knop om de geselecteerde WBS code te verwijderen uit de work breakdown structure. Wanneer de WBS code onderliggende WBS codes heeft in de work breakdown structure, worden deze 'kind' WBS codes eveneens verwijderd.
Inspringen Klik op deze knop om de geselecteerde WBS code in zijn hogerniveau WBS code te laten springen.
Uitspringen Klik op deze knop om de geselecteerde code een kind van de WBS code van de bovenliggende work breakdown structure te maken.
Omhoog Klik op deze knop om de geselecteerde WBS code en de bijbehorende 'kind' WBS codes een positie omhoog te verplaatsen in de work breakdown structure.
Naar beneden Klik op deze knop om de geselecteerde WBS code en de bijbehorende 'kind' WBS codes een positie omlaag te verplaatsen in de work breakdown structure.
Hiërarchie Weergave Klik op deze knop om een dialoogvenster weer te geven dat u kunt gebruiken om de kleuren lettertypes te specificeren die worden gebruikt in de subtitels die verschijnen wanneer u sorteringen/groeperingen maakt die zijn gebaseerd op de work breakdown structure.
... een nieuwe WBS code toevoegen aan de work breakdown structure?
  1. Selecteer de WBS code waaronder u een nieuwe WBS code wilt toevoegen. De code die u selecteert zal de 'hoger-niveau' code van de nieuwe WBS code worden. Bijvoorbeeld, om een nieuwe WBS code direct onder de hoogste-niveau code toe te voegen, selecteer de hoogste-niveau code.
  2. Klik op Toevoegen.
  3. Voer een alfanumeriek id-nummer in voor de code in de WBS code kolom.
  4. Voer een beschrijvende titel in voor de code in de WBS naam kolom.
 ... een bestaande WBS code bewerken?
  1. Klik in de WBS code kolom om het alfanumerieke id-nummer te wijzigen.
  2. Klik in de WBS naam kolom om de beschrijvende titel van de code te wijzigen.
 ... de positie van een WBS code in de work breakdown structure wijzigen?
  • Selecteer een WBS code en klik op Omhoog of Naar beneden om de positie in de work breakdown structure verticaal te wijzigen.
  • Selecteer een WBS code en klik op Inspringen om de geselecteerde code een 'kind' te maken van de WBS code die er in de work breakdown structure boven staat.
  • Selecteer een WBS code en klik op Uitspringen om de geselecteerde code uit zijn hogerniveau WBS code te halen

Indien een WBS code onderliggende WBS codes heeft in de work breakdown structure, worden deze 'kind' WBS codes samen met de geselecteerde WBS code verplaatst.

 ... een WBS code uit de work breakdown structure verwijderen?

Om een WBS code te verwijderen uit de work breakdown structure, selecteert u de code die u wilt verwijderen en klikt u op Verwijderen. Wanneer de WBS code onderliggende WBS codes heeft in de work breakdown structure, worden deze 'kind' WBS codes eveneens verwijderd.

 ... de hiërarchie weergave instellingen voor sorteringen/groeperingen gebaseerd op de work breakdown structure wijzigen?

Klik op Hiërarchie weergave. Het Hiërarchie weergave dialoogvenster verschijnt. U kunt dit gebruiken om de kleuren en lettertypen te specificeren die worden gebruikt in de subtitels die verschijnen wanneer u sorteringen/groeperingen maakt die zijn gebaseerd op de work breakdown structure.
