Selecteer Server dialoogvenster

U gebruikt het Selecteer Server dialoogvenster om de Enterprise server te selecteren waar u het geselecteerde programma naar toe wilt uitchecken. Het dialoogvenster toont een lijst van alle Enterprise servers waartoe u toegang heeft.

Hoe toegang te krijgen tot het Selecteer Server dialoogvenster

Het Selecteer Server dialoogvenster verschijnt wanneer u een programma uitcheckt naar een Enterprise server.

 Verklaring van het veld op het dialoogvenster
Veld Omschrijving
Server lijst Selecteer de Enterprise server waar u het geselecteerde programma naar toe wilt uitchecken.
... de Enterprise server waar u het geselecteerde programma naar toe wilt uitchecken specificeren?
  1. Selecteer de server uit de lijst van alle beschikbare Enterprise servers.
  2. Klik op OK.
