Backstage view - tabblad Nieuw

U gebruikt het Nieuw tabblad van de Backstage view om nieuwe projecten aan te maken.

Als het selectievakje Nieuw project op het tabblad Algemeen van het dialoogvenster Opties is uitgeschakeld, gebruikt u het tabblad Nieuw van de Backstage-view om nieuwe Easyplan-projecten te maken. Merk op dat als het selectievakje Nieuw project op het tabblad Algemeen van het dialoogvenster Opties is ingeschakeld, wanneer u op Bestand en vervolgens op Nieuw klikt, de Nieuw project wizard verschijnt in plaats van het tabblad Nieuw van de Backstage-view.

Beschikbare Sjablonen

Voer een van de onderstaande handelingen uit:
Bestandsnaam

Voer een bestandsnaam in voor het nieuwe project, en eventueel een pad. Wanneer u geen pad opgeeft, wordt het project opgeslagen als een lokaal bestand in de map die is gespecificeerd in het Project pad, op het tabblad Bestandslocaties van het dialoogvenster Opties.
Blader

Klik op deze knop om een ​​ander pad te selecteren, om het project op te slaan op een externe server, om de gebruikersnaam en het wachtwoord te wijzigen, of om indien nodig een ​​werkgroep specificeren.

Wanneer u een nieuw sub-project via de Bladeren knop creëert, kunt u geen projectenjabloon selecteren, omdat het sub-project wordt gecreëerd als een niveau taak binnen het hoofd project.
Titel Voer een beschrijvende naam in voor het project.
Voor Voer de naam van de persoon of organisatie in waarvoor het project wordt aangemaakt.
Door Voer de naam van de persoon of organisatie in die het project aanmaakt.
Start Voer de datum in waarop de vroegste taak in het project start.
Gebruikersnaam Voer indien nodig een naam in als hoofd - administratieve - gebruiker in het project. In de meeste gevallen is het te adviseren om dit ingesteld te laten staan op 'Admin'.
Wachtwoord Voer indien nodig een wachtwoord in waarmee de hoofd - administratieve - gebruiker het project moet openen.
Aanmaken Klik op deze knop om een nieuw project aan te maken waarbij de door u in de bovenstaande velden ingevoerde details worden gebruikt. Wanneer u ervoor heeft gekozen om het project op een bestaand project te baseren, verschijnt het Selecteer Bestaand Project dialoogvenster, waarin u het bestaande project selecteert waarop het nieuwe project gebaseerd is.
