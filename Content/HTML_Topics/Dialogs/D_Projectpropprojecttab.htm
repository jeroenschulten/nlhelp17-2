Dialoogvenster Eigenschappen - tabblad Project

U gebruikt het Project tabblad van het Eigenschappen dialoogvenster om informatie in verhouding tot het huidige project te bekijken en te bewerken. Het Project tabblad verschijnt alleen in projecten in een projectgroep.

Verklaring van de velden op het dialoogvenster
Veld Omschrijving
Projectnaam Voer een beschrijvende naam in voor het project.
Manager Voer de naam in van de persoon die het project beheert.
Startdatum De startdatum van de eerste taak in het project.
Einddatum De einddatum van de laatste taak in het project.
Duur De tijdsperiode van de taak voor het project.
Duur eenheid Selecteer de tijdseenheid waarin u wilt dat de duur wordt getoond.
Geplande start De datum waarop een project moet beginnen. u kunt een geplande startdatum instellen wanneer u niet wilt dat het project begint op de datum die getoond wordt in Startdatum op dit tabblad. U kunt niet zowel een startdatum als een einddatum plannen
Gepland einde De datum waarop het project moet eindigen. U wilt mogelijk een geplande einddatum instellen wanneer u niet wilt dat het project eindigt op de datum die getoond wordt in Einddatum op dit tabblad. U kunt niet zowel een startdatum als een einddatum plannen.
voortgangsdatum De bij benadering datum waarop het project voortgang geboekt heeft. Dit is de datum tot waar voorgang is vergrijsd op de niveautaak van de programmakaart - klik op het globe icoon in de project view om de niveautaak te zien.
Rapportdatum De rapportdatum van de voortgangsperiode die het laatst gebruikt is in het project.
% voltooid Het totale duur percentage voltooid van het project.
Kosten De totale kosten van het project.
Inkomsten De totale inkomsten van het project.
Inspanning De totale hoeveelheid ingeplande inspanning in het project. De totale hoeveelheid aangevraagde inspanning wordt getoond tussen haakjes.
Werk De totale hoeveelheid werk uitgevoerd door de resources in het project.
... een start of einddatum bij het project plannen?

Voer een datum in bij ofwel het Geplande start of het Gepland einde veld. U kunt niet zowel een startdatum als een einddatum plannen.
