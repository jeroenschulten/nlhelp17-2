Backstage view - tabblad Info

U gebruikt het Info tabblad van de Backstage view om de toegang tot het actieve project met een wachtwoord beveiligen en om aan te geven dat bepaalde gebruikers hun toegangsrechten tot het project behouden wanneer het is vergrendeld, om de locatie van de Enterprise server op te geven.

Uitsluitingsinstellingen

Klik op deze knop om het Uitsluitingsinstellingen dialoogvenster weer te geven, dat u kunt gebruiken om te specificeren dat bepaalde gebruikers hun toegangsrechten tot het huidige project behouden wanneer deze is vergrendeld.

Wanneer gebruikers uitgesloten zijn bij een project kunnen ze het project niet bewerken of opslaan, en kunnen nieuwe gebruikers niet bij het project. U zou het handig kunnen vinden dat u, terwijl u veranderingen aanbrengt aan een project, er zeker van bent dat geen enkele andere gebruiker in staat is om tegelijkertijd iets te veranderen aan het project. Merk op dat u alleen gebruikers uit kunt sluiten bij server-based projecten.

 

Deze knop is niet beschikbaar in standalone.

 

Klik hier voor meer informatie.
Server Locatie

Klik op deze knop om het Client/Server Communicatie Details dialoogvenster weer te geven, dat u kunt gebruiken om de locatie van de Enterprise server op te geven, zodat de client er zich aan kan koppelen.

 

Deze knop is niet beschikbaar in standalone.

 

Klik hier voor meer informatie.
... het huidige project met wachtwoorden beveiligen?
  1. Klik op Wachtwoorden. Het Wachtwoorden dialoogvenster verschijnt.
  2. Gebruik dit dialoogvenster uw wachtwoord, of het wachtwoord van een ondergeschikte gebruiker, in te stellen of te wijzigen, en klik vervolgens op OK.
 ... gebruikers specificeren die moeten worden uitgesloten van een project vergrendeling?
  1. Klik op Uitsluitingsinstellingen. Het Uitsluitingsinstellingen dialoogvenster verschijnt.
  2. Schakel de selectievakjes in naast de gebruiker(s) die u wilt uitsluiten van de vergrendeling, klik vervolgens op OK. U kunt nu het project vergrendelen.
 ... de view specificeren die standaard wordt getoond wanneer het project wordt geopend?

Selecteer de view die u initieel wilt weergeven wanneer het project wordt geopend - mits die Project standaard view staat geselecteerd in het Herladen bij openen project veld op het tabblad View van het Opties dialoogvenster - in het Standaard view veld.
