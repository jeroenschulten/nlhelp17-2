Dialoogvenster Opslaan Taak Standaardprofiel

U gebruikt het Opslaan Taak Standaardprofiel dialoogvenster om een naam in te voeren voor een nieuwe set van standaards voor taken. U kunt meerdere profielen voor verschillende sets van standaard taakinstellingen hebben. Voorbeeld: een standaard set om taken te creëren met een bepaalde kalender en code en een andere set om taken te creëren met een bepaalde duur en hoeveelheid werk.

Hoe toegang te krijgen tot het Opslaan Taak Standaardprofiel dialoogvenster

Om toegang te krijgen tot het Opslaan Taak Standaardprofiel dialoogvenster, klikt u op Opslaan als op het Taak tabblad van het Balkkaart Standaards Profielen dialoogvenster.

 Verklaring van de velden op het dialoogvenster
Veld Omschrijving
Dit taak standaard profiel opslaan als Voer een naam in voor de kopie van de set met standaard taak instellingen.
... het geselecteerde taak standaard profiel onder een andere naam opslaan?
  1. Voer een naam in voor de kopie van het taak standaard profiel in het Dit taak standaard profiel opslaan als veld.
  2. Klik op OK.
