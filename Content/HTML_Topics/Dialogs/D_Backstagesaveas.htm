Backstage view - tabblad Opslaan Als

U gebruikt het Opslaan als tabblad van de Backstage view om projecten met een andere bestandsnaam op te slaan, via ofwel standaard bestandsformaat of via een ander bestandsformaat. Wanneer u Enterprise gebruikt, kunt het actieve project ook opslaan in een server-gemounte datasource.

Verklaring van de velden en knoppen op het Opslaan Als tabblad
Veld/Knop Omschrijving
Deze PC Klik op deze knop om het actieve project op te slaan in een map op uw pc of op een toegewezen netwerkstation, met behulp van het standaard -bestandsformaat of een ander bestandsformaat. Er verschijnt een lijst met navigeerbare mappen. Navigeer door de mappenlijst, voer een bestandsnaam in, geef de indeling op waarin u het project wilt opslaan en klik op Opslaan.
Enterprise Klik op deze knop om het actieve project op te slaan in een server-gemounte datasource. Er verschijnt een navigeerbare lijst met datasources en projecten. Navigeer door de datasources en projecten, voer een projectnaam in en klik op Opslaan.
Browse Klik op deze knop om het dialoogvenster Opslaan als weer te geven, waarmee u een alternatieve methode kunt vinden om te zoeken naar de locatie waar u lokale en op de server-gemounte projecten kunt opslaan.
... het actieve project met een andere bestandsnaam in een map op mijn pc of op een toegewezen netwerkstation opslaan?
  1. Klik op Deze PC. Er verschijnt een navigeerbare mappenlijst.
  2. Navigeer door de mappenlijst naar de map waarin u het project wilt opslaan.
  3. Voer een bestandsnaam in waarmee u het project wilt opslaan.
  4. Klik op Opslaan.
 ... het actieve project in een ander bestandsformaat in een map op mijn pc of op een toegewezen netwerkstation opslaan?
  1. Klik op Deze PC. Er verschijnt een navigeerbare mappenlijst.
  2. Navigeer door de mappenlijst naar de map waarin u het project wilt opslaan.
  3. Voer een bestandsnaam in waarmee u het project wilt opslaan.
  4. Selecteer het bestandsformaat waarin u het project wilt opslaan in het veld onder de bestandsnaam.
  5. Klik op Opslaan.
 ... het actieve project in een server-gemounte datasource opslaan?
  1. Klik op Enterprise. Er verschijnt een navigeerbare lijst met datasources en projecten.
  2. Navigeer door de datasources en projecten.
  3. Voer een naam in waarmee u het project wilt opslaan.
  4. Klik op Opslaan.
 ... het Opslaan Als dialoogvenster gebruiken om te specificeren hoe en waar het actieve project op te slaan?
  1. Klik op Bladeren. Het Opslaan Als dialoogvenster verschijnt.
  2. Navigeer naar de locatie waarin u het project wilt opslaan.
  3. Voer een bestandsnaam in voor het project in het Naam veld.
  4. Selecteer het bestandsformaat waarin u het project wilt opslaan in het Opslaan als type veld.
  5. Klik op Opslaan.
  6. Selecteer het project en klik op Open om het te openen.
