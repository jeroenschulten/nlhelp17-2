Selecteer vervangend object dialoogvenster

U gebruikt het Selecteer Vervangend Object dialoogvenster om het object te specificeren waarmee u het Bibliotheek Verkenner object wilt vervangen dat u aan het verwijderen bent. Het object dat u selecteert in dit dialoogvenster zal toegewezen worden aan alles waaraan het object dat u aan het verwijderen bent was toegewezen.

Hoe toegang te krijgen tot het Selecteer Vervangend Object dialoogvenster

Het Selecteer Vervangend Object dialoogvenster verschijnt wanneer u ervoor kiest om een Bibliotheek Verkenner object te verwijderen en te vervangen.
