Dialoogvenster Gebruiker Eigenschappen - tabblad Bibliotheek Toegangsrechten

U gebruikt het Bibliotheek Toegangsrechten tabblad van het Security Groep Eigenschappen dialoogvenster om de rechten aan te geven die de geselecteerde security groep heeft tot folders in de Bibliotheek verkenner. Voor bepaalde bibliotheken kunt u toegangsrechten specificeren voor ieder individueel bibliotheek item; voor andere bibliotheken kunt u alleen toegangsrechten specificeren voor de bibliotheek zelf. U gebruikt dit tabblad ook om de toegangsrechten tot baselines te specificeren.

Als u een Rechten bibliotheek gespecificeerd hebt op het tabblad Algemeen, kunt u deze bibliotheek instellen als alleen-lezen, om te voorkomen dat de leden van de security groep hun toegangsrechten tot de bibliotheek wijzigen.

U kunt dit tabblad ook gebruiken om de resource aan te geven waarvoor de geselecteerde security groep optreedt als Resource Manager in de Time Sheet applicatie.

Verklaring van de velden en knoppen op het dialoogvenster

Het dialoogvenster toont het volgende veld aan de linkerkant:

Veld Omschrijving
Bibliotheek hiërarchie

Een hiërarchische weergave van de bibliotheekobjecten binnen het programma van projecten, en een lijst van de baselines van het programma.

 

Klik op een item in dit veld om het te selecteren. U kunt de om de leden van de security groep toegangsrechten voor het geselecteerde item instellen met de velden aan de rechterkant van het dialoogvenster.

 

Wanneer u rechten vastgelegd hebt voor een item, wordt het item vet weergegeven, tenzij u ingesteld heeft dat het item dezelfde rechten krijgt als het hoger-niveau object ervan.

Het dialoogvenster toont de volgende velden en knoppen voor elk item dat is geselecteerd in het Bibliotheek hiërarchie veld:

Veld/Knop Omschrijving
Lezen Schakel dit selectievakje in om de leden van de security groep leesrechten te geven voor het geselecteerde object.
Aanmaken Schakel dit selectievakje in om de leden van de security groep toe te staan objecten van het geselecteerde type te creëren. Voor baselines geldt dat dit selectievakje de leden van de security groep toestaat om projecten als baselines te importeren.
Verwijderen Schakel dit selectievakje in om de leden van de security groep toe te staan objecten van het geselecteerde type te verwijderen.
Aanpassen Schakel dit selectievakje in om de leden van de security groep toe te staan objecten van het geselecteerde type aan te passen. Voor baselines geldt dat dit selectievakje de leden van de security groep toestaat om data in baselines samen te voegen.
Toewijzen Schakel dit selectievakje in om de leden van de security groep toe te staan objecten van het geselecteerde type toewijzen (toe te wijzen). Niet alle bibliotheekobjecten kunnen toewijzingsrechten hebben.
Toewijzing verwijderen Schakel dit selectievakje in om de leden van de security groep toe te staan toewijzingen bij objecten van het geselecteerde type te verwijderen. Niet alle bibliotheekobjecten kunnen toewijzing verwijderen rechten hebben.
Voortgang goedkeuren Schakel dit selectievakje in om de leden van de security groep toe te staan timesheets goed te keuren van de geselecteerde resource in Time Sheet. Wanneer u dit selectievakje inschakelt, kunnen de leden van de security groep optreden als Resource Manager voor de geselecteerde resource in Time Sheet.
Onbeperkt Klik op deze knop om de leden van de security groep volledige toegang te geven tot het geselecteerde item.
Geen Klik op deze knop om de leden van de security groep geen toegang te geven tot het geselecteerde item.
Zoals hogerniveau Klik op deze knop om de leden van de security groep dezelfde toegangsrechten tot het geselecteerde object te geven als tot het hogerniveau object.
Inkomsten Schakel dit selectievakje in om de leden van de security groep toegang te geven tot inkomsten voor het type object dat is geselecteerd, bijvoorbeeld resources of kosten en inkomsten. De leden van de security groep moeten tenminste lezen rechten hebben voor het geselecteerde object item om toegang te hebben tot de inkomsten ervan. Wanneer de leden van de security groep lezen en schrijven rechten hebben voor het geselecteerde object, hebben zij deze ook voor de inkomsten.
Uitgaven Schakel dit selectievakje in om de leden van de security groep toegang te geven tot uitgaven voor het type object dat is geselecteerd, bijvoorbeeld resources of kosten en inkomsten. De gebruiker moet tenminste lezen rechten hebben voor het geselecteerde object item om toegang te hebben tot de uitgaven ervan. Wanneer de gebruiker lezen en schrijven rechten heeft voor het geselecteerde object, heeft hij deze ook voor uitgaven.
... toegangsrechten voor de Bibliotheek Verkenner mappen en objecten specificeren?
  1. Selecteer de bibliotheekmap of het bibliotheekitem waarvoor u toegangsrechten wilt specificeren.
  2. Gebruik de selectievakjes in de Object Rechten groep om de toegangsrechten voor de geselecteerde bibliotheekobjecten te specificeren: schakel een selectievakje in om toegangsrechten toe te wijzen of schakel een selectievakje uit om toegangsrechten te weigeren.
  3. Specificeer of de leden van de security groep toegangsrechten hebben tot de inkomsten en uitgaven van het geselecteerde object (bijvoorbeeld kosten en inkomsten voor resources) met de selectievakjes in de Toegang voor Velden groep.
 ... toegangsrechten voor de baselines specificeren?
  1. Selecteer de baseline waarvoor u toegangsrechten wilt specificeren, of klik op Baselines om toegangsrechten voor alle baselines te specificeren.
  2. Gebruik de selectievakjes in de Object Rechten groep om de toegangsrechten voor de geselecteerde baseline te specificeren: schakel een selectievakje in om toegangsrechten toe te wijzen of schakel een selectievakje uit om toegangsrechten te weigeren.
  3. Specificeer of de leden van de security groep toegangsrechten hebben tot de baseline inkomsten en uitgaven informatie met de selectievakjes in de Toegang voor Velden groep.
 ... specificeren of de leden van de security groep timesheets kan goedkeuren die verstuurd zijn door resources met behulp van de Time Sheet applicatie?
  1. Selecteer een resource folder of individuele resource.
  2. Schakel het Voortgang Goedkeuren selectievakje in om het de leden van de security groep mogelijk te maken timesheets goed te keuren die worden uitgevoerd door het geselecteerde resource, of schakel het selectievakje uit, om te voorkomen dat de gebruiker de timesheet kan goedkeuren.
