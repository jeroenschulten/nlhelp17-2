Dialoogvenster XML Export Opties

U gebruikt het XML Export Opties dialoogvenster om de manier te specificeren waarop u het huidige project wilt exporteren naar Microsoft Project XML formaat.

Exporteren van een project naar XML formaat stelt u in staat om projectinformatie met collega's te delen die toegang hebben tot Microsoft Project maar geen toegang hebben tot . Let echter op de tekortkomingen van dit bestandsformaat, bepaalde informatie van uw project kan verloren gaan aangezien veel gegevens niet naar XML formaat geëxporteerd kunnen worden.

Hoe toegang te krijgen tot het XML Export Opties dialoogvenster

Het XML Export Opties dialoogvenster verschijnt wanneer u ervoor kiest om een project op te slaan in Microsoft Project XML formaat (*.XML).

 Verklaring van de velden op het dialoogvenster
Veld Omschrijving
Alle Klik op dit keuzerondje wanneer u het volledige programma van projecten wilt exporteren.
Groep Klik op dit keuzerondje wanneer u alleen de huidige groep van het project wilt exporteren.
Voeg vlaggen toe aan taken zonder inkomende links Schakel dit selectievakje in wanneer u een start vlag wilt toevoegen aan alle taken in het geëxporteerde project zonder inkomende links. Dit kunt u doen om te voorkomen dat Microsoft Project automatisch deze taken naar de begindatum van het project verplaatst.
Exporteer inkomsten als negatieve kosten Microsoft Project ondersteunt geen inkomsten, daarom moet u de manier waarop u wilt dat het XML bestand met inkomsten omgaat specificeren. Schakel dit selectievakje in om alle inkomsten in het project om te zetten naar negatieve kosten in het XML bestand, of schakel het selectievakje uit zodat alle inkomsten tegelijkertijd uit het XML bestand verwijderd worden
Exporteer subtitels

Microsoft Project ondersteunt geen subtitels - balken met naam maar zonder taak - daarom moet u de manier waarop u wilt dat het XML bestand met subtitels omgaat specificeren. Schakel dit selectievakje in om alle subtitels om te zetten in verborgen taken die de naam krijgen van de subtitel balk in het XML bestand, of schakel het selectievakje uit om alle subtitels uit te sluiten bij het exporteren naar een XML bestand.

 

Als u ervoor kiest om subtitels te exporteren door ze om te zetten in verborgen taken in Microsoft Project, worden de verborgen taken terug omgezet in subtitels wanneer het XML bestand weer geïmporteerd wordt in . Echter, let er op dat ondanks dat de taken die gebruikt worden om de subtitels weer te geven in Microsoft Project niet zichtbaar zijn, ze wel effect hebben op de bijbehorende samenvattingstaken.
... de manier specificeren waarop ik het huidige project kan exporteren naar Microsoft Project XML formaat?
  1. Klik op het Alle keuzerondje wanneer u het hele bestand wilt exporteren naar een XML bestand, of klik op het Groep keuzerondje wanneer u alleen de huidige groep van het project wilt exporteren.
  2. Schakel het Voeg vlaggen toe aan taken zonder inkomende links selectievakje in om een start vlag toe te voegen aan alle taken zonder inkomende links in het XML bestand. U kunt dit doen om te voorkomen dat Microsoft Project automatisch zulke taken naar de startdatum van het project verplaatst.
  3. Microsoft Project ondersteunt geen inkomsten, daarom moet u de manier waarop u wilt dat het XML bestand met inkomsten omgaat specificeren. Schakel het Export inkomsten als negatieve kosten selectievakje in om alle inkomsten in het project om te zetten naar negatieve kosten in het XML bestand, of schakel het selectievakje uit zodat alle inkomsten tegelijkertijd uit het XML bestand verwijderd worden
  4. Microsoft Project ondersteunt geen subtitels - balken met naam maar zonder taak - daarom moet u de manier waarop u wilt dat het XML bestand met subtitels omgaat specificeren. Schakel het Export subtitels selectievakje in om alle subtitels om te zetten in verborgen taken die de naam krijgen van de subtitel balk in het XML bestand, of schakel het selectievakje uit om alle subtitels uit te sluiten bij het exporteren naar een XML bestand.
  5. Klik OK om het project naar Microsoft Project XML formaat te exporteren.
