Selecteer WBS Code dialoogvenster

U gebruikt het Selecteer WBS Code dialoogvenster om een WBS code vanuit de work breakdown structure van het project aan de op dit moment geselecteerde taak toe te wijzen.

Hoe toegang te krijgen tot het Selecteer WBS Code dialoogvenster

Om toegang te krijgen tot het Selecteer WBS Code dialoogvenster, klikt u op de knop naast het WBS code veld op het Taak tabblad van het Balk en Taak Eigenschappen dialoogvenster.

 Verklaring van de velden op het dialoogvenster
Veld Omschrijving
Zoek

Voer de WBS code in - of het begin van de WBS code - in dit veld om de code te vinden in de work breakdown structure. Wanneer u begint met typen, zijn de WBS codes in het dialoogvenster beperkt tot diegene die beginnen met de tekens die u in typt.

 

Dit is een handige manier om de WBS codes te vinden in grote work breakdown structuren.
Work breakdown structure Dit veld toont de hiërarchische boomstructuur van WBS codes. Selecteer een WBS code en klik op OK om deze toe te wijzen aan de geselecteerde taak.
... de WBS code om toe te wijzen aan de geselecteerde taak specificeren ?

De work breakdown structure, een hiërarchische boomstructuur van WBS codes, wordt weergegeven in het hoofdpaneel van het dialoogvenster. Selecteer een WBS code en klik op OK om deze toe te wijzen aan de geselecteerde taak.

Wanneer de work breakdown structure van het project erg groot is, kan het moeilijk zijn de WBS code die u wilt toewijzen aan de taak direct te vinden. Wanneer u de alfanumerieke identifier van de WBS code die u wilt toewijzen aan de taak kent- of tenminste het begin ervan - kunt u deze vinden in de work breakdown structure door de WBS code - of het begin van de WBS code - in het Zoek veld bovenaan het dialoogvenster. Wanneer u begint te typen in het Zoek veld, worden de WBS codes in het dialoogvenster beperkt tot diegene die beginnen met de tekens die u intypt.
