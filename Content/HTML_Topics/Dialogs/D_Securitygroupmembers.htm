Security Groep Leden dialoogvenster

U gebruikt het Security Groep Leden dialoogvenster om gebruikers toe te wijzen aan security groepen. U selecteert de security groep waaraan u gebruikers toe wilt wijzen, en gebruikt vervolgens de selectievakjes in de gebruikershiërarchie om gebruikers toe te wijzen aan de geselecteerde security groep. U kunt dit dialoogvenster ook gebruiken om gebruikers uit een security groep te halen, door het selecteren van de security groep en de selectievakjes in de gebruikershiërarchie uit te schakelen.

Hoe toegang te krijgen tot het Security Groep Leden dialoogvenster

Om toegang te krijgen tot het Security Groep Leden dialoogvenster, rechtsklik op een security groep in de Bibliotheek Verkenner en selecteer Security Groep Leden.

 Verklaring van de velden op het dialoogvenster
Veld Omschrijving
Security groep

Selecteer de security groep waaraan u gebruikers wilt toewijzen, of waaruit u gebruikers wilt weghalen.

 

Selecteer Geen in dit veld om alleen die gebruikers weer te geven die nog niet aan een security groep in de gebruikershiërarchie zijn toegewezen. Dit is een goede manier van identificeren van de gebruikers die u aan een security groep moet toewijzen.
Voer filter tekst hier in Voer tekst in dit veld om te zoeken naar gebruikers waarvan de naam matcht met de tekst die u invoert. Hierdoor is het eenvoudiger om specifieke gebruikers in dit dialoogvenster in projecten met een groot aantal gebruikers terug te vinden.
Gebruikers hiërarchie

Schakel het selectievakje van een gebruiker in om een gebruiker toe te wijzen aan de geselecteerde security groep, of schakel het selectievakje uit om ze uit de geselecteerde security groep weg te halen.

 

Wanneer u een gebruiker die al van een security groep lid is toewijst aan een andere security groep, wordt deze gebruiker automatisch uit de oude security groep weggehaald.
... gebruikers toewijzen aan security groepen?
  1. Selecteer de security groep waaraan u gebruikers wilt toewijzen, of waaruit u gebruikers wilt weghalen, in het Security groep veld.

    Selecteer Geen in dit veld om alleen die gebruikers weer te geven die nog niet aan een security groep in de gebruikershiërarchie zijn toegewezen. Dit is een goede manier van identificeren van de gebruikers die u aan een security groep moet toewijzen.
  2. Schakel het selectievakje van een gebruiker in om een gebruiker toe te wijzen aan de geselecteerde security groep, of schakel het selectievakje uit om ze uit de geselecteerde security groep weg te halen.
  3. Ga door met het toewijzen van gebruikers aan security groepen, en klik vervolgens op Sluiten wanneer u hiermee klaar bent.
