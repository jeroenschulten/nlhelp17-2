Dialoogvenster Lint Aanpassingen - tabblad Object Bewerken werkbalk

U gebruikt het tabblad Object Bewerken werkbalk van het Lint Aanpassingen dialoogvenster om de opdrachten die zich bevinden op Object Bewerken werkbalk aan te passen. U kunt opdrachten toevoegen aan de Object Bewerken werkbalk, opdrachten verwijderen uit de werkbalk, en de locatie wijzigen van de opdrachten op de werkbalk.

Verklaring van de velden en knoppen op het dialoogvenster
Veld/Knop Omschrijving
Kies Opdrachten van

Selecteer een categorie om de opdrachten te tonen in die categorie in de Opdrachtenlijst direct onder dit veld. Selecteer:

  • Alle Opdrachten om alle beschikbare opdrachten te tonen, ongeacht of ze wel of niet worden getoond op het Lint.
  • Opdrachten niet in het Lint om juist Alle Opdrachten te tonen die op dit moment niet op het Lint staan.
  • Object Bewerken om alle opdrachten te tonen die zouden kunnen worden weergegeven worden op de Object Bewerken werkbalk.
  • Macro's om uw gebruikersmacro's te tonen.
  • Een specifiek Lint tabblad om alle Opdrachten te tonen die op dat specifieke tabblad staan.
Opdrachtenlijst Een lijst met alle beschikbare opdrachten in de op dit moment geselecteerde categorie. Klik op een opdracht om deze te selecteren.
Toevoegen Klik op deze knop om de op dit moment in de Opdrachtenlijst geselecteerde opdracht toe te voegen aan de Object Bewerken werkbalk. De geselecteerde opdracht wordt toegevoegd aan het einde van de Object Bewerken werkbalk Opdrachtenlijst.
Verwijderen Klik op deze knop om de op dit moment in de Object Bewerken werkbalk Opdrachtenlijst geselecteerde opdracht te verwijderen.
Object Bewerken werkbalk Opdrachtenlijst

Een lijst met opdrachten die op dit moment zijn opgenomen in de Object Bewerken werkbalk. Klik op een opdracht om deze te selecteren - ofwel om deze van de Object Bewerken werkbalk te verwijderen, ofwel om haar plaats binnen de werkbalk te veranderen, of (in het geval van macro-commando's) om haar naam en icoontje te veranderen.

 
Klik op deze knop om de geselecteerde opdracht één plaats naar boven te verplaatsen in de Object Bewerken werkbalk Opdrachtenlijst - waardoor ze een plaats naar links zal verplaatsen op de werkbalk.
Klik op deze knop om de geselecteerde opdracht één plaats naar beneden te verplaatsen in de Object Bewerken werkbalk Opdrachtenlijst - waardoor ze een plaats naar rechts zal verplaatsen op de werkbalk.
Wijzigen Klik op deze knop om de naam, omschrijving en icoontje van het op dit moment in de Object Bewerken werkbalk Opdrachtenlijst geselecteerde opdracht te wijzigen.
... een opdracht toevoegen aan de Object Bewerken werkbalk?
  1. Lokaliseer de opdracht die u wilt toevoegen aan de Object Bewerken werkbalk door de juiste categorie te selecteren in het Kies Opdrachten van veld en selecteer de opdracht in de Opdrachtenlijst die er direct onder verschijnt.
  2. Klik op Toevoegen.
 ... een opdracht verwijderen uit de Object Bewerken werkbalk?
  1. Lokaliseer de opdracht die u wilt verwijderen in de Object Bewerken werkbalk Opdrachtenlijst.
  2. Klik op Verwijderen.
 ... de plaats van een opdracht op de Object Bewerken werkbalk veranderen?
  1. Lokaliseer de opdracht die u wilt verplaatsen in de Object Bewerken werkbalk Opdrachtenlijst.
  2. Klik op om de geselecteerde opdracht één plaats naar boven te verplaatsen in de lijst - waardoor ze een plaats naar links zal verplaatsen op de werkbalk - of klik op één plaats naar beneden te verplaatsen - waardoor ze een plaats naar rechts zal verplaatsen op de werkbalk.
 ... de naam, omschrijving en icoontje van een macro-commando op de Object Bewerken werkbalk wijzigen?
  1. Lokaliseer het macro-commando die u wilt wijzigen in de Object Bewerken werkbalk Opdrachtenlijst.
  2. Klik op Wijzigen. Een dialoogvenster verschijnt waarin u de naam, omschrijving en icoontje van het macro-commando kunt wijzigen.
  3. Voer een nieuwe naam en omschrijving in voor het macro-commando in het Weergave naam veld, selecteer desgewenst een ander icoontje en klik op OK.
