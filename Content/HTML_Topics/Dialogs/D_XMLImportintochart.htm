Microsoft Project Importeer Opties dialoogvenster

U gebruikt het Microsoft Project Importeer Opties dialoogvenster om de manier te specificeren waarin u het geselecteerde Microsoft® Project XML of MPP bestand in een kaart of samenvattingsgroep wilt importeren.

Hoe toegang te krijgen tot het Microsoft Project Importeer Opties dialoogvenster

Om toegang te krijgen tot het Microsoft Project Importeer Opties dialoogvenster, rechtsklik op een kaart of samenvattingsgroep in de project view en selecteer Importeer in kaart vanuit Microsoft Project in het menu dat verschijnt.

Schakel dit selectievakje in om

Omdat Microsoft Project zelf geen codebibliotheken ondersteund, worden codebibliotheek toewijzingen in Microsoft Project geregistreerd als tekst-type gebruikersvelden bij de relevante taken en resources. Wanneer u ervoor kiest om codebibliotheken te importeren, worden deze geconverteerd naar codebibliotheken en codebibliotheek items in . Wanneer de namen van elke codebibliotheek of codebibliotheek item dat wordt geïmporteerd overeenkomen met bestaande codebibliotheken of codebibliotheek items op dezelfde locatie in de codebibliotheek hiërarchie, zullen de bestaande codebibliotheek items toegewezen worden aan de geïmporteerde taken of toewijzingen, in plaats van dat er duplicaat codebibliotheken of codebibliotheek items worden gecreëerd.
