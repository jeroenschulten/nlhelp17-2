Dialoogvenster Inloggen bij het project

U kunt het dialoogvenster Inloggen bij het project om in te loggen op een geselecteerde server of project, wanneer de server of het project beschermd wordt door een wachtwoord.

Hoe toegang te krijgen tot het Inloggen bij het project dialoogvenster

Het Inloggen bij het project dialoogvenster verschijnt wanneer u een project probeert te openen dat wachtwoord-beveiligd is, of wanneer u een Enterprise gebruiker bent en de server wachtwoord-beveiligd is. Het Inloggen bij het project dialoogvenster verschijnt ook wanneer u klikt op de Openen als nieuwe Gebruiker knop op het Openen Project dialoogvenster, of op de Aanmaken als nieuwe Gebruiker knop op het Nieuw dialoogvenster.
