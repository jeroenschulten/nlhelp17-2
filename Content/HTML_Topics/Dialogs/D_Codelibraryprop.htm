Dialoogvenster Codebibliotheek Eigenschappen

U gebruikt het Codebibliotheek Eigenschappen dialoogvenster om details van het geselecteerde codebibliotheek te onderhouden. U kunt de naam van de codebibliotheek veranderen, specificeren of het mogelijk is om meerdere codes vanuit de bibliotheek aan individuele objecten toe te wijzen, en specificeren hoe het kritieke pad wordt weergegeven op objecten waaraan een code vanuit de bibliotheek is toegewezen.

Hoe het Codebibliotheek Eigenschappen dialoogvenster op te roepen

Om toegang te krijgen tot het Codebibliotheek Eigenschappen dialoogvenster, rechtsklik op een codebibliotheek in de Bibliotheek Verkenner of de project view en selecteer Eigenschappen.

 Verklaring van de velden op het dialoogvenster
Veld Omschrijving
Naam Voer een beschrijvende naam in om toe te passen op de codebibliotheek.
Maar één code

Schakel dit selectievakje in om te specificeren dat het alleen maar mogelijk is om één enkele code vanuit de codebibliotheek aan elk individueel object in het project toe te wijzen, of schakel dit selectievakje uit om te specificeren dat het ook mogelijk is meer dan één code vanuit de bibliotheek aan elk individueel object toe te wijzen.

Wanneer u dit selectievakje inschakelt voor een codebibliotheek waaraan het voorheen wel was toegestaan om meerdere codes aan elk object toe te wijzen, zullen alle al gemaakte meervoudige toewijzen in stand worden gehouden.
Kritieke stijl

Standaard worden kritieke objecten gemarkeerd door een rode omranding, maar u kunt ook een rode code toewijzen aan een object, en dan is het misschien niet duidelijk welke objecten zich op het kritieke pad bevinden. Selecteer een andere kritieke opmaak in dit veld door een andere dan de rode kleur te kiezen voor kritiek markering.

Bijvoorbeeld, wanneer u rode taakcodes in Risico Factor codebibliotheek wilt gebruiken, kunt u de kritieke weergave van die codebibliotheek instellen als een blauwe stippellijn zodat de kritieke taken met rode codes op de balkkaart nog steeds onderscheiden kunnen worden.
... specificeren of er meer dan één code vanuit de bibliotheek aan individuele objecten kan worden toegewezen?

Schakel het Maar één code selectievakje in om te specificeren dat het alleen maar mogelijk is om één enkele code vanuit de codebibliotheek aan elk individueel object in het project toe te wijzen, of schakel dit selectievakje uit om te specificeren dat het ook mogelijk is meer dan één code vanuit de bibliotheek aan elk individueel object toe te wijzen.

 ... specificeren hoe het kritieke pad weergegeven wordt op objecten waaraan een code vanuit de bibliotheek is toegewezen?

Selecteer een stijl waarmee het kritieke pad weergegeven wordt op objecten waaraan een code vanuit de bibliotheek is toegewezen in het Kritieke stijl veld.
