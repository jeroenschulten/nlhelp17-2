Toon Types in Project View dialoogvenster

U gebruikt het Toon Types in Project View dialoogvenster om de soorten informatie te specificeren die u wilt weergeven in de project view.

Hoe toegang te krijgen tot het Toon Types in de Project View dialoogvenster

Om toegang te krijgen tot het Toon Types in de Project View dialoogvenster, klik op de Toon Types knop op het View tabblad van het Opties dialoogvenster.

 Verklaring van de velden op het dialoogvenster
Veld Omschrijving
Kostengroepen Schakel dit selectievakje in om de kostengroephiërarchie in de project view weer te geven, of schakel het selectievakje uit om kostengroepen te verbergen bij de project view.
Codebibliotheken Schakel dit selectievakje in om de codebibliotheek hiërarchie in de project view weer te geven, of schakel het selectievakje uit om te verbergen codebibliotheken te verbergen bij de project view.
Kalenders Schakel dit selectievakje in om de kalenderhiërarchie in de project view weer te geven, of schakel het selectievakje uit om te verbergen kalenders te verbergen bij de project view.
Resources Schakel dit selectievakje in om de resourceshiërarchie in de project view weer te geven, of schakel het selectievakje uit om te verbergen permanent resources te verbergen bij de project view.
Toewijzingsgroepen Schakel dit selectievakje in om toewijzingsgroepen in de project view weer te geven, of schakel het selectievakje uit om toewijzingsgroepen te verbergen bij de project view.
Hammock taken

Schakel dit selectievakje in om hammock taken in de project view weer te geven, of schakel het selectievakje uit om hammocks te verbergen bij de project view.

 

Klik hier voor meer informatie.
Mijlpaalweergaven

Schakel dit selectievakje in om mijlpaalweergaven in de project view te tonen, schakel het selectievakje uit om mijlpaalweergaven te verbergen bij de project view.

 

Klik hier voor meer informatie.
WBS Schakel dit selectievakje in om de work breakdown structure in de project view weer te geven, of schakel het selectievakje uit om de WBS te verbergen bij de project view.
Samenvattingen Schakel dit selectievakje in om samenvattingsgroepen in de project view om weer te geven, of schakel het selectievakje uit om samenvattingsgroepen te verbergen bij de project view en alleen kaarten te tonen.
Voortgangsperioden Schakel dit selectievakje in om de voortgangsperiodehiërarchie in de project view weer te geven, of schakel het selectievakje uit om voortgangsperioden te verbergen bij de project view.
Verbruiksgoederen Schakel dit selectievakje in om het verbruiksgoederenhiërarchie in de project view weer te geven, of schakel het selectievakje uit om verbruiksgoederen te verbergen bij de project view.
Taak pools

Schakel dit selectievakje in om taak pools in de project view weer te geven, die kunnen worden gebruikt als sjablonen om netwerken van taken te creëren, of schakel het selectievakje uit om taak pools te verbergen bij de project view.

 

Klik hier voor meer informatie.
IFC producten

Schakel dit selectievakje in om de hiërarchie van IFC product zoeksets en IFC product selecties in de project view weer te geven, of schakel het selectievakje uit om IFC product zoeksets en IFC product selecties te verbergen bij de project view.

 

Dit selectievakje verschijnt alleen wanneer u gebruik maakt van . Zie de Help voor meer informatie.
Gearchiveerde kaarten

Schakel dit selectievakje in om de gearchiveerde kaarten en samenvattingsgroepen in de project view weer te geven, of schakel het selectievakje uit om gearchiveerde kaarten en samenvattingsgroepen te verbergen bij de project view. Merk op dat archiveren alleen beschikbaar is in Enterprise.

 

Klik hier voor meer informatie.
... de informatie specificeren die wordt weergegeven in de project view?

Schakel het selectievakje om dat type informatie in de project view te tonen, of schakel het selectievakje uit om dat type informatie uit te sluiten in de project view.
