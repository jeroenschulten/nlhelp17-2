Backstage view - Import/Export tab

U gebruikt het tabblad Import/Export van de Backstage-view om projecten vanuit verschillende bestandsindelingen in te importeren en om projecten vanuit naar verschillende bestandsindelingen te exporteren, zodat uw projecten in andere projectmanagement toepassingen kunnen worden geopend.

Verklaring van de velden en knoppen op het Import/Export tabblad
Veld/Knop Omschrijving
Importeer van Primavera®

Klik op deze knop om een project dat is opgeslagen in XER- of XML-formaat in Primavera P6 te importeren, of om een project dat is opgeslagen in DIR- of STX-formaat in Primavera SureTrak® te importeren in .

Klik hier voor meer informatie.
Exporteer naar Primavera®

Klik op deze knop om het huidige project te exporteren naar een indeling waarin het kan worden geopend in Primavera P6®.

Klik hier voor meer informatie.
Importeer van Microsoft Project®

Klik op deze knop om een project dat is opgeslagen in MPP- of XML-indeling in Microsoft® Project te importeren in .

Klik hier voor meer informatie.
Exporteer naar Microsoft Project®

Klik op deze knop om het huidige project te exporteren naar een indeling waarin het kan worden geopend in Microsoft Project.

Klik hier voor meer informatie.
Exporteer naar HTML

Klik op deze knop om het huidige project naar HTML te exporteren. Wanneer u een project naar HTML exporteert, wordt de tabelinformatie geconverteerd naar een HTML-tabel en kunt u kiezen of u het balkkaart wilt converteren naar een afbeelding voor opname in het HTML-bestand.

Klik hier voor meer informatie.
Exporteer naar CSV

Klik op deze knop om de tabelinformatie uit de huidige view te exporteren naar de Comma Separated Variable (CSV) -indeling, die kan worden gebruikt door applicaties zoals spreadsheets.

Klik hier voor meer informatie.
... een project van Primavera software importeren?
  1. Klik op Importeer van Primavera. Het Primavera XML & XER Importeer Opties dialoogvenster verschijnt.
  2. Gebruik de velden op het dialoogvenster om het project te selecteren dat u wilt importeren en om een aantal belangrijke importopties te speciferen.
  3. Klik op OK.
 ... het huidige project naar een bestandsformaat exporteren waarin het geopend kan worden in Primavera P6?
  1. Klik op Exporteer naar Primavera. Het Primavera XML & XER Exporteer Opties dialoogvenster verschijnt.
  2. Gebruik de velden op het dialoogvenster om het bestandsformaat te specificeren waarnaartoe u het huidige project wilt exporteren en om een aantal belangrijke exportopties te specificeren.
  3. Klik op OK.
 ... een project van Microsoft Project importeren?
  1. Klik op Importeer van Microsoft Project. Het Microsoft Project Importeer Opties dialoogvenster verschijnt.
  2. Gebruik de velden op het dialoogvenster om het project te selecteren dat u wilt importeren en om een aantal belangrijke importopties te speciferen.
  3. Klik op OK.
 ... het huidige project naar een bestandsformaat exporteren waarin het geopend kan worden in Microsoft Project?
  1. Klik op Exporteer naar Microsoft Project. Het Exporteer dialoogvenster verschijnt.
  2. Gebruik het dialoogvenster om een bestandsnaam en locatie te specificeren voor het exportbestand.
  3. Klik op Opslaan. De Microsoft Project Exporteer Wizard verschijnt.
  4. Gebruik de wizard om een aantal belangrijke exportopties te specificeren.
  5. Klik op Voltooien.
 ... het huidige project naar HTML exporteren?
  1. Klik op Exporteer naar HTML. Het Exporteer naar HTML dialoogvenster verschijnt.
  2. Gebruik het dialoogvenster om een bestandsnaam en locatie te specificeren voor het exportbestand en om een aantal belangrijke exportopties te specificeren.
  3. Klik op OK.
 ... tabelinformatie vanuit de huidige view exporteren naar CSV?
  1. Klik op Exporteer naar CSV. Het Exporteer dialoogvenster verschijnt.
  2. Gebruik het dialoogvenster om een bestandsnaam en locatie te specificeren voor het exportbestand.
  3. Klik op Opslaan.
