Filter Wizard - Taak ID (WBN) pagina

U gebruikt de Filter Wizard om de criteria voor filters te maken en te bewerken en om te aan te geven welke items er in een filter opgenomen moeten worden. De Filter Wizard bestaat uit een aantal pagina's. Deze pagina van de Filter Wizard verschijnt wanneer u gekozen hebt om te filteren op de taak ID (WBN) van balken en taken. U gebruikt deze pagina om de taak ID (WBN) code van balken en taken die zijn opgenomen in de gefilterde view te specificeren. U kunt uw filter hoofdlettergevoelig maken.

Verklaring van de velden op het dialoogvenster
Veld Omschrijving
Zoek naar volgend ID

Voer het unieke taak ID in waarop u wilt filteren.

 

U kunt jokertekens (wildcards) gebruiken om een groep van unieke taak ID's op te nemen. Typ bijvoorbeeld ID9* om te zoeken naar balken en taken met unieke taak ID's beginnend met 'ID9'.
Hoofdlettergevoelig Schakel dit selectievakje in om het filter hoofdlettergevoelig te maken, of schakel het selectievakje uit om te zoeken naar de tekst die u hebt ingevoerd, ongeacht de hoofdlettergevoeligheid. Indien u een filter hoofdlettergevoelig maakt, zal zoeken naar taken met een taak ID (WBN) code 'WBN0001' geen taken opleveren met een code 'wbn0001' of 'Wbn0001'.
Reguliere expressies gebruiken in plaats van wildcards

Schakel dit selectievakje in wanneer u wilt zoeken en daarbij reguliere expressies wilt gebruiken in plaats van jokertekens

Vraag steeds opnieuw naar de taak ID (WBN) Schakel dit selectievakje in wanneer u de op te nemen unieke taak ID wilt kunnen specificeren iedere keer dat u het filter toepast, of schakel het selectievakje uit om de gespecificeerde parameters in deze wizard elke keer te gebruiken.
  1. Typ de taak ID (WBN) code die wilt opnemen in de gefilterde view in het veld Zoek naar volgende taak ID's . U kunt wildcards gebruiken om een groep van taak ID (WBN) codes op te nemen. Typ bijvoorbeeld WBN9* om te zoeken naar balken en taken met taak ID (WBN) codes beginnend met 'WBN9'.
  2. Schakel het selectievakje Hoofdlettergevoelig in om het filter hoofdlettergevoelig te maken, of schakel het selectievakje uit om te zoeken naar de tekst die u hebt ingevoerd, ongeacht de letterweergave. Indien u het filter hoofdlettergevoelig maakt, zal zoeken naar taken met een taak ID (WBN) code 'WBN0001' geen taken opleveren met een code 'wbn0001' of 'Wbn0001'.
  3. Selecteer Reguliere expressies gebruiken in plaats van wildcards indien u reguliere expressies, een krachtige tool om met tekst en gegevens om te gaan, in uw filtercriteria wilt gebruiken.
  4. Wanneer u de taak ID (WBN) code wilt kunnen specificeren iedere keer dat u het filter toepast, schakel dan het Vraag me welke Taak ID (WBN) te gebruiken.... selectievakje in.
  5. Klik op Volgende om naar de volgende pagina van de wizard te gaan.
