Dialoogvenster Project Taak ID (WBN) Nummering Definitie

U gebruikt het Project Taak ID (WBN) Nummering Definitie dialoogvenster om aan te geven hoe taak ID (WBN) codes genereert. U kunt taak ID (WBN) codes gebruiken om de taakposities te herkennen binnen de hiërarchie van uw project. kan een uniek taak ID (WBN) code genereren voor elke taak in een project. De taak ID (WBN) code, welke automatisch wordt aangemaakt zodra u een taak maakt, is een tekststring die gebruik maakt van tekens uit een door u bepaalde reeks (bijvoorbeeld, 0-9 of A-Z), met scheidingstekens die de projecthiërarchie aangeven. Bij iedere nieuwe taak die binnen het project wordt aangemaakt, wordt de code opgehoogd met een hoeveelheid die u kunt aangeven. U kunt specificeren of de taak ID (WBN) codes uniek zijn binnen het hele project, binnen een sub-project, of niet uniek zijn.

U kunt de taak ID (WBN) codes van bestaande taken hernummeren. Wanneer u bijvoorbeeld verschillende taak ID (WBN) codes wilt toepassen op alle taken in een specifieke subkaart of een specifieke view. Om dit te kunnen doen, moet u de taken die u wilt hernummeren, selecteren voor u dit dialoogvenster opent. Wanneer u geen taken selecteert, zal de hernummering alleen effect hebben op nieuwe taken. Om het gehele project te hernummeren, klikt u op de Bewerken groep op het Start tabblad, om eerst alles te selecteren.

Hoe toegang te krijgen tot het Project Taak ID (WBN) Nummering Definitie dialoogvenster

Om toegang te krijgen tot het Project Taak ID (WBN) Nummering Definitie dialoogvenster, op het Project tabblad, in de Eigenschappen groep, klikt u op Taak ID (WBN) Nummering.

 Verklaring van de velden op het dialoogvenster
Veld Omschrijving
Taak ID's (WBN) Uitschakelen

Schakel dit selectievakje in om taak ID (WBN) codes in het project uit te schakelen, of schakel het selectievakje uit om de codes in te schakelen.

 

Wanneer u een ongeldige WBN definitie in dit dialoogvenster intypt, zal het selectievakje automatisch selecteren, om te voorkomen dat ongeldige codes aan taken worden toegewezen.
Domein Voer de reeks tekens in dat u wilt laten gebruiken om taak ID (WBN) codes te genereren. Veel gebruikte domeinen zijn 0-9, a-z en A-Z. Het grootste domein dat u kunt specificeren is 0-z, en dit is het domein dat u moet specificeren als een alfanumerieke tekststring de Taak ID (WBN) code vooraf moet gaan.
Startwaarde Voer het teken of de tekens binnen het opgegeven domein in waarmee de taak ID (WBN) nummering moet beginnen. Standaard is dit het eerste teken in het opgegeven domein, bijvoorbeeld 0 of A.
Verhogen Voer het aantal tekens in dat moet worden vrijgehouden tussen een taak ID (WBN) code en de volgende. Als de codes opvolgend moeten zijn, geef dan 1 op in het veld; als u wilt dat de ID's met 2 worden verhoogd, geef dan 2 op. Een verhoogwaarde van 2 zorgt voor het volgende rijtje WBN codes: 1, 3, 5, 7 etc. of A, C, E...
Minimum breedte Voer het minimum aantal cijfers in waaruit iedere code moet bestaan (exclusief de voorloopcode). Codes die kleiner zijn dan de opgegeven breedte worden aangevuld met nullen. Bijvoorbeeld, wanneer een taak ID (WBN) code 12 zou zijn en de minimum breedte is 4, zal de code 0012 zijn.
Padnaam scheidingsteken Specificeer het teken waarmee taak ID (WBN) codes gescheiden worden van een taak ID (WBN) pad, bijvoorbeeld, 1.5.10 of 1/5/10.
Taak scheidingsteken Specificeer het teken waarmee een taak ID (WBN) code gescheiden wordt van zijn pad, bijvoorbeeld 1/5-10.
Taak ID (WBN) voorvoegsel Voer een alfanumerieke tekststring in die u wilt toevoegen als voorvoegsel bij elke taak ID (WBN) code.
Uniek binnen hele bestand Klik op dit keuzerondje om taak ID (WBN) codes uniek te laten zijn binnen het hele bestand.
Uniek binnen samenvattings-/niveautaken Klik op dit keuzerondje om taak ID (WBN) codes uniek te maken binnen niveau- en samenvattingstaken. Dit kan erin resulteren dat hetzelfde taak ID (WBN) gebruikt wordt voor taken in verschillende kaarten binnen hetzelfde project.
Niet uniek Klik op dit keuzerondje om toe te staan dat taak ID (WBN) codes dubbel mogen voorkomen in het bestand.
Taken op dezelfde regel mogen dezelfde WBN delen Schakel dit selectievakje in om aan te geven dat taken op een regel dezelfde taak ID (WBN) code mogen hebben, of schakel het selectievakje uit om verschillende codes te forceren. Als u dit selectievakje selecteert, dan blijft verschillende codes aan de taken op een balk toewijzen, maar nu kunt u ze wel wijzigen in dezelfde code.
Selectie hernummeren Dit veld is alleen ingeschakeld als één of meer taken op dat moment zijn geselecteerd. Schakel dit selectievakje in om de geselecteerde taken te hernummeren met de taak ID (WBN) definitie die u hebt ingevoerd in dit dialoogvenster, of schakel vakje uit wanneer u de taak ID (WBN) definitie alleen wilt toepassen op nieuwe taken.
Binnen geselecteerde samenvattings- /niveautaken hernummeren Dit veld is alleen ingeschakeld als één of meer taken op dit moment zijn geselecteerd. Schakel dit selectievakje in om alleen de taken binnen alle geselecteerde samenvattingstaken of niveautaken te hernummeren, of schakel het selectievakje uit om alleen de taken in de huidige view te hernummeren. Om het hele project te hernummeren selecteert u de taken in de programma kaart en selecteert u deze optie.
Tegenstrijdige taak ID's (WBN) overslaan

Dit veld is alleen ingeschakeld als één of meer taken op dit moment zijn geselecteerd. Schakel dit selectievakje in om tegenstrijdige hernummeringen over te slaan, of schakel het selectievakje uit om hernummering af te breken wanneer een tegenstrijdigheid wordt gevonden.

 

Wanneer de geselecteerde taken geblokkeerde objecten bevatten of wanneer u taak ID (WBN) codes hebt ingevoerd voor samenvattingstaken, niveautaken of hammock taken die conflicteren met de gegenereerde taak ID (WBN) codes, zal de hernummering worden afgebroken tenzij u dit selectievakje selecteert.
... de manier specificeren waarop taak ID (WBN) codes toepast?
  1. U kunt de aanmaak van taak ID (WBN) codes uitschakelen door het WBN uitschakelen selectievakje te selecteren.
  2. Voer de reeks tekens in die u wilt dat gebruikt om taak ID (WBN) codes te genereren in de Domein velden. Veel gebruikte domeinen zijn 0-9, a-z en A-Z. Het grootste domein dat u kunt specificeren is 0-z, en dit is het domein dat u moet specificeren als een alfanumerieke tekststring de Taak ID (WBN) code vooraf moet gaan.
  3. Voer het teken of de tekens binnen het opgegeven domein in waarmee taak ID (WBN) nummering moet beginnen in het Startwaarde veld. Standaard is dit het eerste teken in het opgegeven domein, bijvoorbeeld 0 of A.
  4. Voer het aantal tekens in dat moet worden vrijgehouden tussen een taak ID (WBN) code en de volgende in het Verhogen veld. Als de codes opvolgend moeten zijn, geef dan 1 op in het veld; als u wilt dat de ID's met 2 worden verhoogd, geef dan 2 op. Een verhoogwaarde van 2 zorgt voor het volgende rijtje WBN codes: 1, 3, 5, 7 etc. of A, C, E...
  5. Voer het minimum aantal cijfers in waaruit iedere code moet bestaan (exclusief de voorloopcode) in het Minimum breedte veld. Codes die kleiner zijn dan de opgegeven breedte worden aangevuld met nullen. Bijvoorbeeld, wanneer een taak ID (WBN) code 12 zou zijn en de minimum breedte is 4, zal de code 0012 zijn.
  6. Voer het teken in waarmee taak ID (WBN) codes gescheiden worden wanneer ze worden weergegeven in een work breakdown pad in het Padnaam scheidingsteken veld.
  7. Voer het teken in waarmee taak ID (WBN) codes gescheiden worden van hun pad in het Taak scheidingsteken veld.
  8. Voer een alfanumerieke tekststring of tekens in die u als voorloopcode wilt toevoegen aan elk taak ID (WBN) code in het Taak ID (WBN) Voorvoegsel veld.
  9. In de Uniek groep, selecteert u of u wilt dat de taak ID (WBN) codes uniek zijn door het hele project, uniek binnen niveau- en samenvattingstaken, of niet uniek.
  10. Schakel het Taken op dezelfde regel mogen dezelfde WBN delen selectievakje in om toe te staan dat taken op dezelfde regel dezelfde taak ID (WBN) code kunnen krijgen, of schakel het selectievakje uit om verschillende codes te forceren. Wanneer u dit selectievakje inschakelt zal taken op dezelfde regel verschillende codes toewijzen, maar nu kunt u ze wel wijzigen in dezelfde code.
  11. De Hernummering groep is alleen ingeschakeld als één of meer taken op dit moment zijn geselecteerd. U gebruikt deze groep om de manier te specificeren waarop u de taak ID (WBN) codes van de geselecteerde taken wilt hernummeren:
    • Schakel het Selectie hernummeren selectievakje in om de geselecteerde taken te hernummeren met de taak ID (WBN) definitie die u in dit dialoogvenster heeft ingevoerd, of schakel het selectievakje uit om de nieuwe definitie alleen toe te passen op nieuwe taken.
    • Schakel het Binnen geselecteerde samenvattings-/niveautaken hernummeren selectievakje in om alleen de taken binnen een geselecteerde samenvattingstaak of niveautaak te hernummeren, of schakel het selectievakje uit om alleen de taken in de huidige view te hernummeren. Om het hele project te hernummeren selecteert u de taken in de programma kaart en selecteert u deze optie.
    • Wanneer de geselecteerde taken geblokkeerde objecten bevatten of wanneer u taak ID (WBN) codes heeft ingevoerd voor samenvattingstaken, niveautaken of hammock taken die tegenstrijdig zijn met de gegenereerde taak ID (WBN) codes, zal de hernummering afbreken, tenzij u het Tegenstrijdige taak ID's (WBN) overslaan selectievakje kiest.
