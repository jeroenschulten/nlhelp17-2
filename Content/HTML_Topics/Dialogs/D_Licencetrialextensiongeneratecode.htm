Wizard Verlenging Probeerperiode - Genereer Verlenging Probeerperiode Aanvraagcode pagina

U gebruikt de Verlenging Probeerperiode Wizard om de licentie voor uw probeerperiode van uw Elecosoft software te verlengen. De Verlenging Probeerperiode Wizard bestaat uit een aantal stappen. Elke stap wordt duidelijk uitgelegd in de schermen van de wizard.

Deze pagina van de Verlenging Probeerperiode Wizard toont informatie die gevraagd wordt door Elecosoft BV om u van een licentiecode te kunnen voorzien - een 'verlenging probeerperiode aanvraag code'. U moet de deze code vanaf deze computer sturen naar Elecosoft BV Support. U kunt dit doen per e-mail, of wanneer uw computer geen e-mail heeft, door gebruik te maken van een andere methode, bijvoorbeeld een floppy disk.

Klik op e-mail om de verlenging probeerperiode aanvraag code over te brengen naar een e-mailbericht, of klik op Kopiëren om de aanvraag te kopiëren naar het Windows® Klembord. U kunt dan de e-mail naar de andere computer sturen, of breng de verlenging probeerperiode aanvraag code over in een TXT bestand op een floppy disk.

Wanneer u de verlenging probeerperiode aanvraag code naar Elecosoft hebt gestuurd, zal de Supportafdeling u een licentiecode sturen, die u kunt gebruiken om uw probeerperiode te verlengen.

Stuur de verlenging probeerperiode aanvraag code naar Elecosoft Support. Klik vervolgens op Volgende om naar de volgende pagina van de licentie wizard te gaan.
