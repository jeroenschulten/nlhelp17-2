Dialoogvenster Security Groep Eigenschappen - tabblad Codebibliotheek Rechten

U gebruikt het Codebibliotheek Toegangsrechten tab van het Security Groep Eigenschappen dialoogvenster om de toegangsrechten te specificeren die leden geselecteerde security groep hebben tot de security bibliotheek die u selecteerde op het tabblad Algemeen van het dialoogvenster. U kunt toegangsrechten specificeren tot die codebibliotheek, of tot de individuele codes binnen de bibliotheek, in aanvulling op de project hiërarchie-gebaseerde toegangsrechten die u specificeert op het Kaart Toegangsrechten tabblad. Dit betekent dat u de leden van een security groep die enkel maar alleen-lezen toegang tot een project hebben, volledige toegangsrechten kunnen hebben tot objecten met een specifieke code, waar die ook in de project hiërarchie voorkomen.

Hoe toegang te krijgen tot het Codebibliotheek Rechten tabblad van het Security Groep Eigenschappen dialoogvenster

Om toegang te krijgen tot het Codebibliotheek Rechten tabblad van het Security Groep Eigenschappen dialoogvenster, specificeer een codebibliotheek om te gaan fungeren als security bibliotheek op het tabblad Algemeen van het Security Groep Eigenschappen dialoogvenster, en klik vervolgens op Eigenschappen.

 Verklaring van de velden en knoppen op het dialoogvenster

Het dialoogvenster toont het volgende veld aan de linkerkant:

Veld Omschrijving
Code bibliotheek hiërarchie

Een hiërarchische weergave van de codebibliotheek die u geselecteerd heeft om te laten fungeren als security bibliotheek.

U kunt toegangsrechten tot de gehele codebibliotheek, of tot individuele codebibliotheekitems binnen de bibliotheek specificeren.

Klik op de codebibliotheek, of op een codebibliotheekitem in dit veld om deze te selecteren. U kunt dan de security groep toegangsrechten tot de geselecteerde code configureren met de velden rechts van het dialoogvenster

Wanneer u rechten voor een codebibliotheekitem heeft ingesteld, wordt dit codebibliotheekitem vet weergegeven, tenzij u voor dit niveau dezelfde rechten hebt ingesteld als voor het niveau daarboven

Het dialoogvenster toont de volgende velden en knoppen die verwijzen naar elke code die op dit moment in het Code bibliotheek hiërarchie veld geselecteerd is:

Veld/Knop Omschrijving
Lezen Schakel dit selectievakje in om de leden van de security groep rechten te geven om balkkaart objecten waaraan de geselecteerde code is toegewezen, ongeacht hun toegangsrechten in de projecthiërarchie. Leden van de security groep krijgen automatisch Leesrechten tot objecten paneer Creëren, Verwijderen of Aanpassen rechten staan ingesteld.
Aanmaken Schakel dit selectievakje in om de leden van de security groep rechten te geven om balkkaart objecten waaraan de geselecteerde code is toegewezen te creëren, ongeacht hun toegangsrechten in de projecthiërarchie.
Verwijderen Schakel dit selectievakje in om de leden van de security groep rechten te geven om balkkaart objecten waaraan de geselecteerde code is toegewezen te verwijderen, ongeacht hun toegangsrechten in de projecthiërarchie.
Bewerken Schakel dit selectievakje in om de leden van de security groep rechten te geven om balkkaart objecten waaraan de geselecteerde code is toegewezen aan te passen, ongeacht hun toegangsrechten in de projecthiërarchie.
Onbeperkt Klik op deze knop om de leden van de security groep volledige toegang te geven tot balkkaart objecten waaraan de geselecteerde code is toegewezen, ongeacht hun toegangsrechten in de projecthiërarchie.
Geen Klik op deze knop om de leden van de security groep geen toegang te geven tot balkkaart objecten waaraan de geselecteerde code is toegewezen, ongeacht hun toegangsrechten in de projecthiërarchie.
Alle als Hogerniveau Klik op deze knop om de leden van de security groep dezelfde toegangsrechten te geven tot de geselecteerde code als ze al hebben tot de hogerniveaucode ervan, ongeacht hun toegangsrechten in de projecthiërarchie.
Invoeren voortgang Schakel dit selectievakje in om de leden van de security groep toestemming te geven voortgang op taken en toewijzingen waaraan de geselecteerde code is toegewezen in te voeren, ongeacht hun toegangsrechten in de projecthiërarchie.
Verwijder voortgang Schakel dit selectievakje in om de leden van de security groep toestemming te geven om bij voltooide gedeelten van taken en toewijzingen waaraan de geselecteerde code is toegewezen te verwijderen, ongeacht hun toegangsrechten in de projecthiërarchie.
Verplaats kaart Schakel dit selectievakje in om de leden van de security groep toestemming te geven om kaarten waaraan de geselecteerde code is toegewezen door de projecthiërarchie te verplaatsen, ongeacht hun toegangsrechten in de projecthiërarchie.
Kopieer Schakel dit selectievakje in om de leden van de security groep toestemming te geven om objecten waaraan de geselecteerde code is toegewezen te kopiëren, ongeacht hun toegangsrechten in de projecthiërarchie.
Beschikbaar in Project Viewer Schakel dit selectievakje in om de kaarten waaraan de geselecteerde code is toegewezen beschikbaar te maken wanneer een lid van de security groep het project in Elecosofts gratis viewersoftware - Project Viewer - ongeacht hun toegangsrechten in de projecthiërarchie.
Inkomsten Schakel dit selectievakje in om de leden van de security groep bij objecten waaraan de geselecteerde code is toegewezen toegang te geven tot inkomstenhoeveelheden, ongeacht hun toegangsrechten in de projecthiërarchie.
Uitgaven Schakel dit selectievakje in om de leden van de security groep bij objecten waaraan de geselecteerde code is toegewezen toegang te geven tot Uitgavenhoeveelheden, ongeacht hun toegangsrechten in de projecthiërarchie.
Speling Schakel dit selectievakje in om de leden van de security groep bij objecten waaraan de geselecteerde code is toegewezen toegang te geven spelingswaarden, ongeacht hun toegangsrechten in de projecthiërarchie.
... toegangsrechten specificeren voor de geselecteerde security codebibliotheek?
  1. Selecteer de code waarvoor u toegangsrechten wilt specificeren - of selecteer de codebibliotheek zelf wanneer u dezelfde toegangsrechten aan alle codes binnen de bibliotheek wilt geven.
  2. Gebruik de selectievakjes om de toegangsrechten te specificeren voor de balkkaart objecten waaraan de geselecteerde code is toegewezen, ongeacht de toegangsrechten in de projecthiërarchie van de security groep: schakel een selectievakje in om de toegangsrechten toe te wijzen of schakel een selectievakje uit om de toegangsrechten te beletten.
