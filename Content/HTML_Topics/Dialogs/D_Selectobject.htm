Selecteer Object dialoogvenster

U gebruikt het Selecteer Object dialoogvenster om een specifiek project object te selecteren om op te nemen in een formule - een object waarvan u wilt dat de informatie in een veld getoond wordt, of een object dat u als functie parameter wilt gebruiken, afhankelijk van de manier waarop u het dialoogvenster heeft benaderd

Hoe toegang te krijgen tot het Selecteer Object dialoogvenster

Om toegang te krijgen tot het Selecteer Object dialoogvenster, klikt u op Object of Veld en selecteert u een veld uit het keuzemenu in het Formule Eigenschappen dialoogvenster.

Gebruik het Object veld aan de linkerkant van het dialoogvenster om het type object te selecteren waarnaar u formule wilt laten verwijzen. De velden aan de rechterkant van het dialoogvenster veranderen afhankelijk van het type object dat u selecteert. Gebruik de velden aan de rechterkant van het dialoogvenster om de exacte objecten die u wilt selecteren te specificeren.

Standaard object

Selecteer dit object type om een veld te laten verwijzen naar elk object waar een tabelcel standaard naar verwijst, bijvoorbeeld, een taak op een taakregel, of een toewijzing op een allocatieregel.

 

Merk op dat dit object type alleen verschijnt wanneer u dit dialoogvenster gebruikt om een object te selecteren waarover u een veld informatie wilt laten weergeven: het verschijnt niet wanneer u op Object klikt in het Formule Eigenschappen dialoogvenster.

... het object specificeren dat ik wil gebruiken als een functie parameter?
  1. Gebruik het Object veld aan de linkerkant van het dialoogvenster om het type object te selecteren waar naar u de formule wilt laten verwijzen. De velden aan de rechterkant van het dialoogvenster veranderen afhankelijk van het type object dat u selecteer.
  2. Gebruik de velden aan de rechterkant van het dialoogvenster om de exacte objecten die u wilt selecteren te specificeren.
  3. Wanneer u wilt dat het geselecteerde object moet worden ontleend aan een baseline in plaats van aan de huidige project-data, specificeer dan een baseline in het Project of baseline veld.
  4. Klik op OK om terug te keren naar het Formule Eigenschappen dialoogvenster.
 ... het object specificeren waar naartoe een formule veld zou moeten verwijzen?
  1. Gebruik het Object veld aan de linkerkant van het dialoogvenster om het type object te selecteren waarnaar u het veld wilt laten verwijzen. Selecteer Standaard object om het veld te laten verwijzen naar elk object waar naar een tabelcel standaard verwijst, bijvoorbeeld, een taak op een taakregel, of een toewijzing op een allocatieregel. De velden aan de rechterkant van het dialoogvenster veranderen afhankelijk van het type object dat u selecteert.
  2. Gebruik de velden aan de rechterkant van het dialoogvenster om de exacte objecten waar naar de formule moet verwijzen te specificeren.
  3. Wanneer u wilt dat het geselecteerde object moet worden ontleend aan een baseline in plaats van aan de huidige project-data, specificeer dan een baseline in het Project of baseline veld.
  4. Gebruik de Veld parameters velden om alle aanvullende parameters bij het geselecteerde veld te specificeren, zoals een datumdomein, of dat u het veld wilt laten verwijzen naar Geplande, Aanvraag of Alle toewijzing types.
  5. Klik op OK om terug te keren naar het Formule Eigenschappen dialoogvenster.
