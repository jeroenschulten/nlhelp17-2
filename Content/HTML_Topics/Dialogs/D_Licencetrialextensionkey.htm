Wizard Verlenging Probeerperiode - Licentie Code pagina

U gebruikt de Verlenging Probeerperiode Wizard om uw eerste probeer licentie van Elecosoft software te verlengen. De Verlenging Probeerperiode Wizard bestaat uit een aantal stappen. Elke stap wordt duidelijk uitgelegd in de schermen van de wizard.

U gebruikt deze pagina van de Verlenging Probeerperiode Wizard om de licentiecode die u hebt ontvangen van Elecosoft BV in de wizard te plakken.

Wanneer u de licentiecode hebt ontvangen van Elecosoft BV, kopieert u deze naar het Windows® Klembord klikt u op Plakken om de code in deze pagina van de wizard te plakken. Klik vervolgens op Einde. De probeerlicentie is nu op deze computer verlengd.
