Eigenschappen dialoogvenster - tabblad Algemeen

U gebruikt het tabblad Algemeen van een dialoogvenster Eigenschappen gebruiken om algemene details van het geselecteerde object te bekijken. Deze informatie is vooral nuttig als u een Enterprise hebt draaien, omdat het tabblad de gebruiker toont waarvoor het object vergrendeld is.

Hoe toegang te krijgen tot het Algemeen tabblad of een dialoogvenster

Het tabblad Algemeen van elk Eigenschappen dialoogvenster geeft gelijksoortige informatie weer. Als u deze informatie niet wilt gebruiken, kunt u het tabblad Algemeen van alle Eigenschappen dialoogvensters verbergen, door het selectievakje Toon Algemene Informatie op het tabblad View in het dialoogvenster Opties uit te schakelen.

 Verklaring van de velden op het dialoogvenster
Veld Omschrijving
Object ID Het interne identificerende nummer van het geselecteerde object.
Object type Het type geselecteerd object, bijvoorbeeld Taak of kostentoewijzing.
Laatst gewijzigd door De naam van de gebruiker die het laatst het geselecteerde object gewijzigd heeft.
Laatst gewijzigd op De datum en de tijd waarop de geselecteerde object het laatst was gewijzigd.
Vergrendeld aan De naam van de gebruiker aan wie het geselecteerde object op dit moment is vergrendeld, d.w.z. de gebruiker die op dit moment het object aan het wijzigen is.
