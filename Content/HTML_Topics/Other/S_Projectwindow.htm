Projectvenster

Een geopend project heeft een projectvenster, dat ook geopend blijft als alle andere vensters op een project gesloten zijn. U kunt het projectvenster gebruiken om een project te activeren wanneer er geen andere vensters van geopend zijn.

Het projectvenster bevat een project structuur. Klik op een view in de project structuur om een venster op het project te openen. Voorbeeld: klik op een kaart pictogram om een balkkaart view van het project te openen.

De projectstructuur in het Projectvenster heeft niet alle functionaliteit van de projectstructuur in de hoofdview van het project.
