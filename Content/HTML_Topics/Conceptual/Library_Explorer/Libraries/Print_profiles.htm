Afdrukprofielen

U kunt de eigenschappen van een afdrukprofiel wijzigen via het bewerken van de opties van het Afdrukprofielen Eigenschappen dialoogvenster. Om toegang te krijgen tot de help bij elk tabblad van het Afdrukprofielen Eigenschappen dialoogvenster, druk op F1 bij het bekijken van het tabblad.
