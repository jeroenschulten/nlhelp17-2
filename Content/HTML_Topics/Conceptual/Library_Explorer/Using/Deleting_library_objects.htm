Verwijderen van bibliotheekobjecten

Om een object of map binnen de Bibliotheek verkenner te verwijderen, voer een van de volgende acties uit:

  • Rechtsklik op het object of de map, selecteer vervolgens Verwijderen.
  • Selecteer het object of de map, druk vervolgens op DEL.

Geen enkele van de standaard bibliotheekobjecten kunnen verwijderd worden.
