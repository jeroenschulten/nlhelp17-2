Objecten selecteren in de Bibliotheek verkenner

Om een object te selecteren in de lijst view van de Bibliotheek verkenner klikt u op het object. CTRL -klik kunt u gebruiken om meer dan één object te selecteren.

Om een aaneensluitend blok van objecten te selecteren, klikt u op het eerste object, vervolgens toetst u SHIFT -klik op het laatste object. Het eerste en het laatste object en alle tussenliggende objecten worden geselecteerd.

Wanneer u een map selecteert in de structuur view, zijn de objecten in de lijst view niet langer geselecteerd.

Wanneer u gebruik maakt van Enterprise wordt een sleutel pictogram gebruikt om aan te geven of een bibliotheek object is geblokkeerd door u of een andere gebruiker. Het sleutelpictogram is goudkleurig als u het object zelf blokkeert en grijs als het is geblokkeerd door een andere gebruiker Het sleutelpictogram is weergegeven op het pictogram van het object.

Voorbeeld:

geeft een niet geblokkeerd permanent hulpmiddel weer.

geeft een door u geblokkeerd permanent hulpmiddel weer.

geeft een permanent hulpmiddel geblokkeerd door een andere gebruiker weer.
