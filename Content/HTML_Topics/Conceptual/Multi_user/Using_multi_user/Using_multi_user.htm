gebruiken in een multi-user enterprise omgeving

is ontworpen om effectief gebruikt te kunnen worden in een multi-user enterprise omgeving. In plaats van exclusieve rechten geven op het gedeelte van het project waar u de gebruiker gegevens wilt bewerken, kunnen gebruikers het gehele project gelijktijdig gebruiken (binnen hun toegangsrechten ) en altijd de nieuwste informatie op het scherm zien. Bijvoorbeeld, als twee of meer gebruikers dezelfde taak op het beeldscherm zien staan, en één van de gebruikers wijzigt de taak, dan zien de andere gebruikers het scherm vernieuwen op het moment dat de wijziging wordt doorgevoerd.

De sleutel tot het gebruik van is 'object vergrendeling'. Wanneer een gebruiker een object selecteert of bewerkt, dan wordt dat object vergrendeld voor de andere gebruikers, deze kunnen het nu nog wel bekijken maar niet bewerken. Wanneer de eerste gebruiker de veranderingen opslaat of annuleert, dan zal het object weer vrijkomen voor de andere gebruikers, en deze kunnen het dan selecteren om te gaan bewerken. Dit ontwerp verzekert dat alle gebruikers, inclusief gebruikers op afstand die het project bewerken met een modemverbinding, een gelijkwaardige toegang tot het project hebben en altijd de laatste gegevens kunnen zien.
