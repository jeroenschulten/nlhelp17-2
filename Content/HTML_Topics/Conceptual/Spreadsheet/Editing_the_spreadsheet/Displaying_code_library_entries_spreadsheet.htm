Codebibliotheekitems weergeven in de tabel

Kiest u ervoor om een code element te tonen in de tabel, dan wordt standaard de naam van de code getoond. Het is ook mogelijk om alternatieve informatie van een code element te tonen. U kunt kiezen om de volgende dingen te laten zien:

  • De inhoud van een gebruikersveld dat is gekoppeld aan een code element.
  • De korte naam van een code element (Korte Naam veld).
  • Het alternatieve nummer (ID) van het code element (Alt Id veld).
  • De datum waarop het code-element voor het laatst veranderd is (Laatst Bewerkt Datum veld).

U kunt deze velden van een code element wijzigen met behulp van de kolommen in de Bibliotheek verkenner.

U kunt ook specificeren of u het pad van het geselecteerde codebibliotheekitem wilt weergeven (d.w.z. de positie van de code binnen de codebibliotheek hiërarchie); Indien u kiest om het codebibliotheek zoekpad weer te geven, kunt u een teken in te voeren waarmee u de verschillende niveaus van de hiërarchie wilt afbakenen, bijvoorbeeld een komma of een koppelteken.

Om het code-element weer te geven in de tabel:

  1. Klik met de rechtermuisknop op een kolomkop en kies vervolgens de opdracht Tabel Definitie. Het dialoogvenster Tabel Definitie Eigenschappen verschijnt.
  2. Lokaliseer de tabelkolom die u wilt bewerken, en vervolgens klikt u op de eerste rij om de balk of taak attribuut te wijzigen, of u klikt op de tweede rij om het toewijzing attribuut te wijzigen. Een hiërarchische drop-down lijst van attributen wordt geactiveerd.
  3. Selecteer het Code Object en vervolgens de Code Bibliotheek waarvan u de elementen wilt tonen in de kolommen. U selecteert Balk.<codebibliotheek-naam> om informatie te tonen die gerelateerd is aan de balk, of taak. <codebibliotheek-naam> om informatie te tonen die gerelateerd is aan de taak.
  4. Voer de codebibliotheek informatie die u wilt weergeven in het Toon veld veld:
    • U typt de naam van het gebruikersveld in, om de inhoud weer te geven.
    • U typt Korte Naam om de korte naam van de codebibliotheek weer te geven.
    • U typt Alt ID om de alternatieve nummering (ID) van het code-element weer te geven.
    • U typt Laatst Bewerkt Datum om de datum te tonen waarop het Code-element het laatst veranderd is.
  5. Gebruik het Toon zoekpad veld om te specificeren of u het pad van het geselecteerde codebibliotheekitem wilt weergeven (d.w.z. de positie van de code binnen de codebibliotheek hiërarchie); Indien u kiest om het codebibliotheek zoekpad weer te geven, gebruikt u het Pad scheidingsteken veld om een teken in te voeren waarmee u de verschillende niveaus van de hiërarchie wilt afbakenen, bijvoorbeeld een komma of een koppelteken.
