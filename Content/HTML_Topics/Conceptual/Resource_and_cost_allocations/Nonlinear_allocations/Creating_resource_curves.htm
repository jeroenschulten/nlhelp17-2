Creëren van een nieuwe resource curve

U kunt resource curves toewijzen

Om een nieuwe resource curve te creëren:

  1. Open de Bibliotheek Verkenner.
  2. Klik op Resource Curve in de boomstructuur view.
  3. Rechtsklik op een leeg gebied in de lijst view en selecteer Nieuwe Resource Curve. Er verschijnt een nieuwe resource curve in de lijst view met de cursor knipperend in de Naam kolom.
  4. Type een naam voor de resource curve, en klik vervolgens ergens anders om dit te deselecteren.
  5. Rechtsklik op de resource curve en selecteer Eigenschappen. Het Resource Curve Eigenschappen dialoogvenster verschijnt.
  6. De grafische representatie van de resource curve wordt bijgewerkt om het aantal punten te tonen dat u ingevoerd heeft.

Creëren van resource curves van individuele toewijzingsprofielen

U kunt ook resource curves van individuele toewijzingsprofielen creëren die u opgezet heeft voor resourcetoewijzingen. U zou dit kunnen doen wanneer u een individueel toewijzingsprofiel voor een resourcetoewijzing heeft opgezet en vervolgens besluit dat u deze wilt kunnen toepassen - als een resource curve - op andere toewijzingen.

Om een resource curve vanuit een individueel toewijzingsprofiel te creëren:

  1. Rechtsklik op de resource waarvan u het profiel wilt opslaan als een resource curve in de Bibliotheek Verkenner en selecteer Eigenschappen. Het Permanente Resource Eigenschappen dialoogvenster verschijnt.
  2. Klik op het Toewijzing tabblad.
