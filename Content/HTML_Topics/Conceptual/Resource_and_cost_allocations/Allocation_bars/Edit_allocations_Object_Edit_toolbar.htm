Toewijzingsbalken bewerken met de werkbalk Object bewerken

U kunt balken, taken en toewijzingen bewerken met de werkbalk Object Bewerken.

Wanneer u een toewijzing heeft geselecteerd, kunt u:

  • Voortgang toewijzen aan de toewijzing door een voltooid percentage in te typen in het veld ; u kunt geen voortgang toewijzen aan aanvraag toewijzingen.
  • De duur veranderen door de waarde in te bewerken; u kunt de duur van een gemodelleerde toewijzing niet bewerken die Duur als te berekenen parameter heeft of de variabele parameter Geen.
  • Verplaats een toewijzing door de datum te bewerken in de velden of , of door te klikken op en .
  • Wijs een code toe aan de toewijzing door een code te selecteren uit het veld . Om een code uit de geselecteerde bibliotheek te selecteren, klikt u op het rolmenu. Om een code te selecteren uit een andere bibliotheek, klikt u met uw rechtermuisknop op het rolmenu en selecteert u de gewenste codebibliotheek. Vervolgens klikt u weer op het rolmenu om de code te selecteren. De weergave van toewijzingsbalken hoeft niet te veranderen afhankelijk van uw huidige scherm instellingen. U kunt meer dan één code uit elke bibliotheek toewijzen.
