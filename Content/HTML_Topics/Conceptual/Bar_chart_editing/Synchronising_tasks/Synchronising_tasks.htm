Taken synchroniseren

In bepaalde omstandigheden wilt u misschien dat twee taken altijd tegelijk starten of eindigen, zelfs als het logische pad toelaat dat de ene taak eerder begint dan de ander.

Om de start van twee taken gelijk te laten vallen tekent u een link van de start van de ene taak naar de start van de andere taak. Daarna tekent u een link van de start van de tweede taak terug naar de start van de eerste taak, zodat de twee links boven op elkaar komen:

Als de startdatum van één van beide taken wordt uitgesteld, wordt ook de startdatum van de andere taak uitgesteld, zodat ze gesynchroniseerd blijven.

U kunt ook het einde van één taak synchroniseren met de start van een andere taak, en u kunt positieve/negatieve wachttijd op deze links instellen, om bijvoorbeeld het midden van een taak te synchroniseren met de start van een ander taak.
