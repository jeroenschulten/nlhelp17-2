Aangeven in welke volgorde codebibliotheek sorteringen/groeperingen voorkomen

Als u taken sorteert/groepeert op een codebibliotheek, worden taken gesorteerd/gegroepeerd volgens de code die u hebt toegekend uit de codebibliotheek. De codes staan standaard op alfabetische volgorde. Voorbeeld: als een codebibliotheek drie codes bevat, Londen, Parijs en New York, en u sorteert op die codebibliotheek, dan zullen de taken worden getoond in de volgorde:

  1. Taken met de code Londen.
  2. Taken met de code New York.
  3. Taken met de code Parijs.
  4. Taken zonder code uit deze codebibliotheek.

Als u codebibliotheek sorteringen/groeperingen niet op alfabetische volgorde wilt sorteren, kunt u de volgorde waarop elke code in een bibliotheek verschijnt, bepalen met een sortering. U kunt dit doen door te klikken op het Sortering/groepering op - Sorteervolgorde keuzerondje op het tabblad View van het Opties dialoogvenster. Vervolgens voert u bij het Sorteervolgorde veld in het Codebibliotheek Item Eigenschappen dialoogvenster een getal in bij elke code dat de volgorde definieert waarin de codes binnen de huidige bibliotheek zouden moeten worden gesorteerd/gegroepeerd. Codes met een lager getal verschijnen eerder in een gesorteerde/gegroepeerde view dan codes met een hoger getal.

Om codebibliotheek sorteringen/groeperingen goed te laten werken moet u de passende getallen voor elk item in een codebibliotheek in het Sorteervolgorde veld invoeren. Bijvoorbeeld, wanneer u de codes in de hierboven genoemde codebibliotheek wilde sorteren/groeperen in de volgende volgorde - Parijs, Londen, New York - zou u 10 voor Parijs, 20 voor Londen en 30 voor New York kunnen invoeren.

U kunt ook de getallen gebruiken die u heeft ingevoerd in het Sorteer volgorde veld voor codebibliotheekitems om de volgorde te bepalen waarin codebibliotheekitems verschijnen in de legenda op de afdrukken. Om de items in de legenda volgens hun sorteervolgorde te ordenen schakelt u het Gebruik codebibliotheek sorteervolgorde in legenda selectievakje in op de Legenda's sectie in de boomstructuur in het tabblad Opmaak van het Afdrukken dialoogvenster.
