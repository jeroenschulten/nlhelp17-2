Specificeren of dubbele items bij meervoudige toewijzingen getoond worden in gesorteerde/gegroepeerde views

Voor ieder sorteer/groepeer criterium dat verwijst naar een toewijzing, kunt u specificeren of items met meerdere codes, kosten of resource toewijzingen meerdere keren verschijnen in een gesorteerde/gegroepeerde view (een keer voor iedere unieke toewijzing waarde of toegewezen code) of slechts eenmaal, afhankelijk van de eerste relevante toewijzing die overeenkomt met de sorteer/groepeer criteria.

Bijvoorbeeld, wanneer u taken sorteert/groepeert op de codes die er aan zijn toegewezen vanuit een bepaalde codebibliotheek, kunt u specificeren of taken waaraan meer dan een code is toegewezen meerdere keren verschijnen - eenmaal voor elke code die er aan is toegewezen - of eenmaal, voor de eerste code die aan de taak is toegewezen.

Illustratie

De eerste illustratie hieronder toont twee taken waaraan codes zijn toegewezen vanuit een codebibliotheek, en een waaraan geen codes zijn toegewezen. Er zijn geen codes toegewezen aan taak 1. Code 'B' is toegewezen aan taak 2. Beide codes 'A' en 'B' uit dezelfde bibliotheek zijn toegewezen aan taak 3; merk op dat code 'A' eerder was toegewezen aan deze taak dan code 'B'. Wanneer deze taken gesorteerd/gegroepeerd zouden zijn in aflopende volgorde naar de naam van de eraan toegewezen codes, waarbij dubbele items niet worden getoond wanneer er meerdere toewijzingen zijn, worden ze als in onderstaande afbeelding weergegeven - taak A verschijnt slechts eenmaal in de gesorteerde/gegroepeerde view, in de lijst onder code 'A', de eerste code die er aan was toegewezen:

Indien de taken worden gesorteerd/gegroepeerd in oplopende volgorde naar de naam van de codes, waarbij dubbele items getoond worden in geval van meerdere toewijzingen, worden ze getoond als onderstaand - taak A verschijnt tweemaal in de gesorteerde/gegroepeerde view, eenmaal onder elke code die eraan is toegewezen:

Om te specificeren dat items met meerdere toewijzingen meerdere keren verschijnen in een gesorteerde/gegroepeerde view, schakelt u het Toon dubbele items voor meerdere toewijzingen keuzevakje bij het bedoelde criterium in op het Sorteer/Groepeer Eigenschappen dialoogvenster.

Merk op dat dit keuzevakje alleen verschijnt voor sorteer/groepeer criteria die verwijzen naar een bepaald soort toewijzingen - bijvoorbeeld, code bibliotheek toewijzingen aan taken of balken, of kosten- of resource toewijzingen aan taken.
