Specificeren van standaard views voor gebruikers

U kunt een standaard view specificeren voor elke gebruiker, en specificeren dat deze view zou moeten worden geopend wanneer de gebruiker een project opent.

Om een standaard view voor een gebruiker te specificeren:

  1. Rechtsklik op de gebruiker in de Bibliotheek Verkenner, selecteer Eigenschappen. Het Gebruiker Eigenschappen dialoogvenster verschijnt.
  2. Selecteer de view waarvan u wilt dat die als standaard view wordt gebruikt wanneer de gebruiker het project opent in het Standaard view veld.
  3. Klik op OK om uw wijzigingen op te slaan.

Om te specificeren dat de Gebruiker standaard view moet verschijnen wanneer het project wordt geopend:

  1. Klik in het tabblad Bestand, klik vervolgens op Opties. Het Opties dialoogvenster verschijnt.
  2. Klik op het tabblad View.
  3. Selecteer de Gebruiker standaard view in het Herladen bij openen project veld.
  4. Klik op OK om uw wijzigingen op te slaan.
