Kopiëren van een view naar een ander project

U kunt views vanuit het ene project naar een ander project kopiëren. Dit kan handig zijn wanneer u een view heeft aangemaakt in een project die ook u wilt gaan gebruiken in ander project zonder dat u de view attributen, filter, sortering/groepering en tabel in het andere project opnieuw op hoeft te zetten. Wanneer u een view vanaf het ene project naar een ander project kopieert, kunt u specificeren of de geassocieerde objecten, zoals de filter, de sortering/groepering en tabel van de view, met de view worden meegekopieerd naar het bestemmingsproject.

Om een view vanuit het ene project naar een ander project te kunnen kopiëren, moeten beide projecten in geopend zijn.

Om een view vanuit het ene project naar een ander project te kopiëren:

  1. Open het project met de view die u wilt kopiëren, en het project waarnaartoe u de view wilt kopiëren.
  2. In het project die de view bevat die u wilt kopiëren, klik op het View tabblad, in de Views groep, op de View dropdown en selecteer Meer Views. Het Meer Views dialoogvenster verschijnt.
  3. Selecteer de view die u wilt kopiëren en klik op Kopiëren Naar. Het Kopieer View naar Project dialoogvenster verschijnt.

De geselecteerde view is nu gekopieerd naar het bestemmingsproject.
