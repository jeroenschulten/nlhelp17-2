De views specificeren om te herladen wanneer een project heropend wordt

U kunt specificeren welke views moeten worden herladen en weergegeven worden wanneer een project heropend wordt. U kunt kiezen om terug te zetten, ofwel:

Om te specificeren welke views zouden moeten worden herladen wanneer een project opnieuw geopend wordt:

  1. Klik op het tabblad Bestand, klik vervolgens op Opties. Het Opties dialoogvenster verschijnt.
  2. Klik op het tabblad View.
  3. Selecteer de view - of views - die zouden moeten worden herladen wanneer een project opnieuw geopend wordt in het Herladen bij openen project veld.

    Wanneer u ofwel de Gebruiker standaard view ofwel de Project standaard view selecteert, kunt u klikken op de Bewerken knop om deze views te veranderen.
  4. Klik op Sluiten om uw wijzigingen op te slaan.
Opmerkingen over het herladen van views wanneer een project wordt heropend
  • Wanneer meer dan een voorkomen van een view geopend is terwijl een project wordt afgesloten, zal alleen het laatste opgeslagen voorkomen van de view herladen worden.
  • Views die worden weergegeven in de 'voorgangers en opvolgers' of 'pad naar taak' modus, views met standalone histogrammen en Line of Balance grafieken en views die waren geopend om taken in een linklus te identificeren of de taken op een bepaald punt in een histogram grafiek worden niet herladen wanneer een project heropend wordt.
  • Wanneer u specificeert dat ofwel de Gebruiker standaard view ofwel de Project standaard view zou moeten worden herladen wanneer een project opnieuw geopend wordt, maar de geselecteerde view niet is gespecificeerd, zal de laatste actieve view in plaats daarvan worden herladen.
  • Wanneer u specificeert dat een gefilterde view zou moeten worden herladen wanneer een project opnieuw geopend wordt en het filter aan de gebruiker vraagt hoe het filter moet worden toegepast, zal het filter dat nu ook weer vragen wanneer het project heropend wordt.
