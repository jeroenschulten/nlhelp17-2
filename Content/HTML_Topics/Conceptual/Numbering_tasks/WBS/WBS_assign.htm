WBS codes toewijzen aan taken

Wanneer u eenmaal een work breakdown structure heeft opgezet, kunt u de WBS codes in de work breakdown structure toewijzen aan de taken in een project. U kunt dit doen door het Selecteer WBS Code dialoogvenster te gebruiken of door de WBS codes aan te klikken en te slepen vanuit de project view.

Om een WBS code toe te wijzen aan een taak met het Selecteer WBS Code dialoogvenster:

  1. Rechtsklik op de taak waaraan u een WBS code wilt toewijzen en selecteer Eigenschappen. Het Balk en Taak Eigenschappen dialoogvenster verschijnt. U kunt ook het Taak tabblad van de eigenschappen view tonen.
  2. Op het Taak tabblad van het dialoogvenster, klikt u op de knop naast het WBS code veld. Het Selecteer WBS Code dialoogvenster verschijnt.

    De work breakdown structure, een hiërarchische boomstructuur van WBS codes, wordt getoond in het hoofdpaneel van het dialoogvenster.
  3. Selecteer een WBS code en klik op OK om deze toe te wijzen aan de geselecteerde taak.

    Wanneer de work breakdown structure van het project erg groot is, kan het moeilijk zijn de WBS code die u aan de taak wilt toewijzen direct te vinden. Indien u de alfanumerieke ID weet van de WBS code die u wilt toewijzen aan de taak - of tenminste beginkarakters ervan - kunt u de WBS code in de work breakdown structure terugvinden door deze WBS code - of het begin van deze WBS code - in te vullen in het Zoek veld bovenaan het dialoogvenster. Wanneer u begint te typen in het Zoek veld, worden de WBS codes in het dialoogvenster beperkt tot die WBS codes die beginnen met de karakters die u intypt.

Om een WBS code door klikken en slepen vanuit de project view toe te wijzen aan een taak:

  1. Navigeer door de WBS hiërarchie in de project view tot u de WBS code vindt die u wilt toewijzen aan een taak.
  2. Klik, en sleep de WBS code naar een taak in de balkkaart.

Merk op dat WBS codes automatisch worden toegewezen aan nieuwe taken wanneer u een nieuwe taak toevoegt aan een view die is gesorteerd door een WBS attribuut.
