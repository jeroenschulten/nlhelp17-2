Een work breakdown structure opzetten

Om een work breakdown structure op te zetten:

  1. Klik in het tabblad Project, in de Eigenschappen groep, op Work Breakdown Structure. Het Work Breakdown Structure dialoogvenster verschijnt.

    De work breakdown structure, een hiërarchische boomstructuur van WBS codes, wordt getoond in het hoofdpaneel van het dialoogvenster. Nieuwe projecten krijgen een eerste WBS code, waarbij de WBS code genomen wordt van de korte naam van het project en de WBS naam genomen wordt uit de lange (gewone) naam van het project. U kunt de separator specificeren die verschijnt tussen elk element van het WBS pad door het Pad separator veld op het Opmaak tabblad van het dialoogvenster Opties te gebruiken.
  2. Om een nieuwe WBS code toe te voegen selecteert u de WBS code waaronder u een nieuwe code wilt toevoegen en klikt u op Toevoegen. De code die u selecteert zal de 'hogerniveau' code van de nieuwe WBS code worden. Bijvoorbeeld, wanneer u een nieuwe WBS code wilt maken direct onder de hoogste-niveau code, selecteert u de hoogste-niveau code en klikt u op Toevoegen.
  3. Voer een alfanumerieke ID in voor de code, in de WBS code kolom.
  4. Voer een omschrijving in voor de code in de WBS naam kolom.
  5. Ga indien nodig door met het toevoegen van WBS codes aan de work breakdown structure.

U kunt de positie van de WBS codes in de work breakdown structuur wijzigen:

  • Selecteer een WBS code en klik op Omhoog of Naar beneden om de positie in de work breakdown structure verticaal te wijzigen.
  • Selecteer een WBS code en klik op Inspringen om de geselecteerde code een 'kind' te maken van de WBS code die er in de work breakdown structure boven staat.
  • Selecteer een WBS code en klik op Uitspringen om de geselecteerde code uit zijn hogerniveau WBS code te halen.

Indien een WBS code onderliggende WBS codes heeft in de work breakdown structure, worden deze 'kind' WBS codes samen met de geselecteerde WBS code verplaatst.

Om een WBS code te verwijderen uit de work breakdown structure, selecteert u de code die u wilt verwijderen en klikt u op Verwijderen. Wanneer de WBS code onderliggende WBS codes heeft in de work breakdown structure, worden deze 'kind' WBS codes eveneens verwijderd.

De work breakdown structure van een project wordt getoond in de project view. U kunt in de project view op dezelfde wijze door de hiërarchie van de breakdown structuur navigeren als in de project hiërarchie van kaarten en samenvattingsgroepen. U kunt specificeren of iedere WBS code in de work breakdown structure verschijnt met haar WBS code of haar WBS naam door de Toon WBS op keuzerondjes op het View tabblad van het Opties dialoogvenster te gebruiken.

Wanneer u een work breakdown structure in een project heeft opgezet, kunt u WBS codes op de taken in het project toepassen om deze taken te definiëren en te groeperen. Merk op dat WBS codes automatisch worden toegewezen aan nieuwe taken wanneer u een nieuwe taak toevoegt aan een view die is gesorteerd op een WBS attribuut.

Hiërarchie weergave instellingen voor een work breakdown structuur specificeren

U kunt de hiërarchie weergave instellingen voor een work breakdown structure specificeren. Deze worden gebruikt om de sorteer/groepeer criteria aan te geven, wanneer u sorteringen/groeperingen maakt en toepast die zijn gebaseerd op een WBS attribuut om de subtitels te kleuren die verschijnen boven iedere gesorteerde groep van taken. Bijvoorbeeld, wanneer u op WBS code sorteert/groepeert, zullen de subtitels de WBS code laten zien die is toegewezen aan iedere gesorteerde groep van taken.

U kunt een andere subtitel weergave specificeren voor elk niveau van de WBS hiërarchie Bijvoorbeeld, wanneer de hiërarchie van een work breakdown structure zes niveaus omvat, kunt u zes verschillende subtitel weergaves definiëren: één voor elk niveau.

Om de hiërarchie weergave instellingen voor een work breakdown structure te specificeren:

  1. Klik op Hiërarchie Weergave in het Work Breakdown Structure dialoogvenster. Het Hiërarchie Weergave Eigenschappen dialoogvenster verschijnt. U kunt ook toegang verkrijgen tot dit dialoogvenster door op een element van de work breakdown structure in de project view te rechtsklikken en dan Weergave te selecteren.

    Elke regel in het raster in dit dialoogvenster vertegenwoordigt een niveau van de WBS hiërarchie Bijvoorbeeld, de weergave instellingen die u definieert voor de eerste regel worden toegepast op het eerste niveau van de work breakdown structure, de instellingen die u definieert voor de tweede regel worden toegepast op het tweede niveau van de WBS, en zo verder.
  2. Klik op Toevoegen om een nieuwe regel toe te voegen aan het raster.
  3. Klik in iedere kolom van het raster om de letterkleur, de achtergrondkleur en de letterinstellingen te specificeren die moeten worden toegepast om subtitels te sorteren voor elk niveau van de WBS.
  4. Klik op OK om het dialoogvenster te sluiten en terug te keren naar het Work Breakdown Structure dialoogvenster.
