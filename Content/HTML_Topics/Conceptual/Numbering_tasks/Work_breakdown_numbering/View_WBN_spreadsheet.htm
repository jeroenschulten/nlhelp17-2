WBN taak ID's in de tabel bekijken

Wanneer u taken op de balkkaart tekent, worden voor elke taak automatisch taak ID (WBN) codes gegenereerd. Deze codes moeten in de tabel worden weergegeven, als u ze wilt zien.

Om de taak ID (WBN) codes weer te geven, maakt u een nieuwe tabel kolom aan dat het taak ID (WBN), of de taak ID (WBN) pad naam bevat. taak ID (WBN) pad naam laat de taak ID (WBN) code van de bovenliggende taken zien, zodat de taakhiërarchie van het project te zien is.

taak ID (WBN) codes in de tabel instellen
  1. Klik met de rechtermuisknop op een kolomkop en kies vervolgens de opdracht Tabel Definitie. Het dialoogvenster Tabel Definitie Eigenschappen verschijnt.
  2. De kolommen in de huidige tabel worden weergegeven in de kolom Veld. Er zijn twee rijen informatie voor elke kolom, omdat een kolom zowel een balk of taak eigenschap als een toewijzing eigenschap kan weergeven.
  3. Klik op Toevoegen. Een nieuwe tabelkolom is nu toegevoegd.
  4. Zoek de nieuwe tabel kolom en klik op de eerste rij ervan.
  5. Klik op het + symbool naast Identificatie, en selecteer daarna taak ID (WBN), of taak ID (WBN) pad naam.
  6. Gebruik de knoppen Omhoog en Omlaag om de kolommen in de tabel te verplaatsen naar de gewenste positie.
  7. Klik op Sluiten.

Naast de weergave in de tabel, kunt u de taak ID (WBN) codes ook weergeven in een notitie, en kunt u ze zien en bewerken op het tabblad Taak in het dialoogvenster Balk en Taak Eigenschappen. U kunt taak ID (WBN) codes ook weergegeven en bewerken in het taak ID (WBN) besturingselement dat u kunt toevoegen aan de Object Bewerken werkbalk.

Taak ID (WBN) codes worden gegenereerd met de standaard taak ID (WBN) definitie totdat u dat verandert. De standaard instellingen voor het genereren van taak ID (WBN) codes zullen er ongeveer uitzien als in de onderstaande illustratie. De taak ID (WBN) pad naam bestaat uit de code van alle bovenliggende taken, en toont daarmee het niveau van de taak in de hiërarchie. In dit voorbeeld heeft de Design taak een taak ID (WBN) code van 1, de Software taak heeft een taak ID (WBN) code van 5 en de taken binnen de Software subkaart zijn doorgenummerd vanuit andere taken in het project.

U kunt de standaard taak ID (WBN) code bewerken, en meer beschrijvende codes gebruiken zoals in de illustratie hieronder. In de hieronder getoonde illustratie is de taak ID (WBN) code van de Design taak veranderd naar DES, de taak ID (WBN) code van Software naar SW en de taak ID (WBN) codes binnen de subkaart Software zijn opnieuw genummerd vanaf 01 zoals de taak ID (WBN) definitie aangeeft.

Taak ID (WBN) codes worden alleen aangemaakt voor niveautaken en samenvattingstaken wanneer deze is aangemaakt door een aanvankelijk gewone taak met behulp van de opdracht 'veranderen in' te wijzigen in een samenvattingstaak of niveautaak. Taak ID (WBN) codes worden niet gegenereerd voor hammocks. U kunt taak ID (WBN) codes voor niveautaken, samenvattingstaken en hammocks direct intypen in de tabel of in het dialoogvenster Balk en Taak Eigenschappen .
