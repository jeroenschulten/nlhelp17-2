Opzetten van WBN taak ID's

Een taak ID (WBN) code in het is een gegenereerde tekststring, meestal een nummer. U kunt bij het aanpassen van de taak ID (WBN) definitie zelf instellen welke karakterreeksen er gebruikt moeten worden voor de gegenereerde tekststring, de startwaarde, de ophoogwaarde en andere opties.

Als u alle taak ID (WBN) codes van bestaande taken wilt hernummeren, dan moeten die taken eerst worden geselecteerd voordat de taak ID (WBN) definitie wordt aangepast. Bijvoorbeeld, u wilt verschillende taak ID (WBN) codes toepassen op alle taken binnen een specifieke subkaart of binnen een specifieke view. Om alles binnen het gehele project te hernummeren, zorg dan dat de programma kaart (het hoogste niveau) zichtbaar is en kies de opdracht Alles selecteren uit het menu Bewerken voordat de taak ID (WBN) definitie wordt aangepast. Als u geen taken selecteert, dan zijn de wijzigingen aan de taak ID (WBN) definitie alleen van toepassing op nieuwe taken.

Om taak ID (WBN) codes op te zetten:

  1. Klik in het tabblad Project, in de Eigenschappen groep, op Taak ID (WBN) Nummering. Het dialoogvenster Project Taak ID (WBN) Nummering Definitie verschijnt.
  2. Het aanmaken van taak ID (WBN) codes kan worden uitgeschakeld met het selectievakje Taak ID's (WBN) uitschakelen. Als u een ongeldige taak ID (WBN) definitie in dit dialoogvenster intypt, dan zal het selectievakje automatisch selecteren, om te voorkomen dat ongeldige codes aan taken worden toegewezen.
  3. Typ in de velden onder Domein het domein van karakters in, dat het programma mag gebruiken om de taak ID (WBN) codes te genereren. Veelgebruikte domeinen zijn 0-9, a-z en A-Z. Het grootste domein dat u kunt specificeren is 0-z, en dit is het domein dat u moet specificeren als een alfanumerieke tekststring de taak ID (WBN) code vooraf moet gaan.
  4. Typ in het veld Startwaarde het karakter of de karakters binnen het opgegeven domein in, waarmee de taak ID (WBN) nummering moet beginnen. Standaard is dit het eerste karakter van het opgegeven domein, bijvoorbeeld een 0 of een A.
  5. Typ in het veld Verhogen het aantal karakters dat moet worden vrijgehouden tussen één WBN code en de volgende taak ID (WBN) code. Als de codes opvolgend moeten zijn, geef dan 1 op in het veld; als u wilt dat de ID's met 2 worden verhoogd, geef dan 2 op. Een verhoogwaarde van 2 zorgt voor het volgende rijtje taak ID (WBN) codes: 1, 3, 5, 7 etc. of A, C, E...
  6. Geef in het veld Minimum breedte de minimale hoeveelheid posities op van de code (exclusief de voorloopcode). Codes die smaller zijn dan de opgeven breedte worden opgevuld met nullen. Bijvoorbeeld, als een taak ID (WBN) code 12 zou zijn, maar de minimale breedte staat op 4, dan wordt de code 0012.
  7. In het veld Padnaam scheidingsteken voert u het karakter in waarmee de verschillende taak ID (WBN) codes van taken gescheiden worden bij de weergave in een work breakdown pad.
  8. In het veld Taak scheidingsteken voert u het karakter in waarmee de verschillende taak ID (WBN) codes worden gescheiden in het zoekpad.
  9. In de groep Uniek kunt u aangeven of de codes uniek binnen het hele bestand moeten zijn, of alleen uniek binnen de samenvattings-/niveautaak of totaal niet uniek zijn.
  10. Voer in het Taak ID (WBN) voorvoegsel veld een alfanumerieke reeks van tekens in dat u wilt gebruiken als voorvoegsel bij elke taak ID (WBN) code.
  11. Schakel het selectievakje Taken op dezelfde regel mogen dezelfde codes delen in, om aan te geven dat taken op dezelfde regel dezelfde taak ID (WBN) code mogen hebben, of schakel het selectievakje uit om te forceren dat taken verschillende taak ID (WBN) codes moeten zijn. Als u dit selectievakje selecteert, dan blijft verschillende codes aan de taken op een balk toewijzen, maar nu kunt u ze wel wijzigen in dezelfde code.
  12. Het groepsveld Hernummering is alleen ingeschakeld als er op dat moment één of meer taken zijn geselecteerd. U gebruikt deze groep om aan te geven op welke manier de hernummering van de taak ID (WBN) codes op de geselecteerde taken moet gebeuren.
    • Schakel het selectievakje Selectie hernummeren in, om alleen de geselecteerde taken te hernummeren met de opgegeven taak ID (WBN) definitie die u hebt ingetypt in het dialoogvenster, of schakel het selectievakje uit om de definitie alleen voor nieuwe taken te gebruiken.
    • Schakel het selectievakje Binnen geselecteerde samenvattings-/niveautaak hernummeren in, om alleen de taken binnen de geselecteerde samenvattingstaak of niveautaak te hernummeren, of schakel het selectievakje uit om alleen de taken in de huidige view te hernummeren. Om een geheel project te hernummeren, selecteer de taken in het programma en selecteer deze optie.
    • Schakel het selectievakje Tegenstrijdige Taak ID (WBN) overslaan in om te voorkomen dat objecten hernummert die zouden kunnen conflicteren met objecten buiten de door hernummeren veranderde groep, of schakel het selectievakje uit om het mogelijk te maken om elk van deze objecten te hernummeren, zelfs als dat kan resulteren in het hebben van dezelfde nummers bij twee of meer objecten.
  13. Klik op OK om de taak ID (WBN) veranderingen te bevestigen.

Als u ervoor kiest om de work breakdown te hernummeren, dan kan het even duren voordat het scherm wordt vernieuwd en de work breakdown nummers zijn gewijzigd.
