Specificeren welke informatie wordt weergegeven in de project view

U kunt de soorten informatie specificeren, bijvoorbeeld kalenders, codes en samenvattingstaken, die u wilt weergeven in de project view via het Toon Types in de Project View dialoogvenster.

Om de types informatie die wordt weergegeven in de project view te specificeren:

  1. Klik op het tabblad Bestand, klik vervolgens op Opties. Het Opties dialoogvenster verschijnt.
  2. Klik op het View tabblad.
  3. Klik op de Toon types knop. Het Toon Types in de Project View dialoogvenster verschijnt. Dit dialoogvenster bevat een selectievakje voor elk type informatie dat u kunt kiezen om weer te geven in de project view.
  4. Schakel het selectievakje in van een item om dat type informatie in de project view weer te geven, of schakel het selectievakje uit om dat type informatie uit te sluiten bij de project view weergave. Bijvoorbeeld, om de verbruiksgoederen resources hiërarchie weer te geven, schakel het Verbruiksgoederen resources selectievakje in; om samenvattingstaken te verbergen, schakel het Samenvattingen selectievakje uit.
  5. Klik op OK het dialoogvenster te sluiten en terug te keren naar het Opties dialoogvenster.
  6. Klik op Sluiten om het Opties dialoogvenster te sluiten.
