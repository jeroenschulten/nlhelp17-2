De opmaak van codes in de project view tonen

U kunt kiezen om de opmaak van elk codebibliotheek item naast de naam weer te geven wanneer deze verschijnt in de project view en in de Object Bewerken werkbalk. Het weergeven van de opmaak van codebibliotheekitems in de project view vergemakkelijkt het voor u om te controleren dat u de juiste code aan de objecten in de balkkaart.

Wanneer u kiest om de opmaak van codes in de project view te tonen, wordt hun opmaak ook getoond in het besturingselement in de Object Bewerken werkbalk.

Om de opmaak van codes in de project view te tonen:

  1. Klik op het tabblad Bestand, klik vervolgens op Opties. Het Opties dialoogvenster verschijnt.
  2. Klik op het tabblad View.
  3. Schakel het Toon codebibliotheek opmaak selectievakje in om de opmaak van codes in de project view te tonen, of schakel het selectievakje uit om alleen hun naam te tonen.
