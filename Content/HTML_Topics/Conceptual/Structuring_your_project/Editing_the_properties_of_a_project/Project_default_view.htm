Specificeren van een standaard view voor een project

U kunt een standaard view voor een project instellen, en aangeven dat deze view moet verschijnen wanneer een gebruiker het project opent.

Om een standaard view voor een project in te stellen:

  1. Klik in het tabblad Bestand op Eigenschappen, klik vervolgens op Geavanceerde Eigenschappen. Het Eigenschappen dialoogvenster verschijnt.
  2. Klik op het tabblad Eigenschappen.
  3. Selecteer de view waarvan u wilt dat die als standaard view wordt gebruikt wanneer het project opent in het Standaard view veld.
  4. Klik op OK om uw wijzigingen op te slaan.

Om te specificeren dat de Project standaard view moet verschijnen wanneer het project wordt geopend:

  1. Klik in het tabblad Bestand, klik vervolgens op Opties. Het Opties dialoogvenster verschijnt.
  2. Klik op het tabblad View.
  3. Selecteer de Project standaard view in het Herladen bij openen project veld.
  4. Klik op OK om uw wijzigingen op te slaan.
