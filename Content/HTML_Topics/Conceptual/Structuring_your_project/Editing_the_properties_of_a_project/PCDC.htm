Een Vereiste Project Contract Datum (PCD) bepalen

Deze PCD functionaliteit is vervangen door de mogelijkheid om een projectdeadline datum op te geven voor een project en deze datum te gebruiken als deadline bij het herberekenen. Klik hier voor meer informatie. Indien gewenst kunt u voor een project een Vereiste Project Contract Datum aangeven, ook wel zachte project einddatum genoemd. Een PCD is een alternatieve datum voor spelingsberekeningen wanneer een project wordt herberekend. Bij een gewone herberekening, is dit de datum aan het eind van de laatste taak van het project, maar u kunt een PCD aangeven die afwijkt van deze datum. Een PCD is een einddatum voor alle taken in het project, dus elke taak die de PCD overschrijdt heeft bij herberekenen negatieve speling (d.w.z. ze worden uiterst-kritiek).

Een Vereiste Project Contract Datum (PCD) invoeren:

  1. klik op het tabblad Bestand, klik vervolgens op Eigenschappen, klik dan op Geavanceerde Eigenschappen. het Eigenschappen dialoogvenster verschijnt.
  2. Klik op het tabblad Velden.
  3. Typ in de kolom Tekststring van een lege regel de tekst 'PCD'.
  4. Typ de Vereiste Project Contract Datum in de kolom Datum/Tijd van dezelfde regel.
  5. Klik op Sluiten.

Om de Vereiste Project Contract Datum van een project te verwijderen, kiest u het tabblad Velden in het dialoogvenster Eigenschappen en verwijdert u de datum uit de PCD regel.
