Voortgang kopiëren van het ene project naar een ander project

U kunt de voortgang dat geregistreerd is in het ene project kopiëren naar een ander project.

Om de voortgang die geregistreerd is in het ene project kopiëren naar een ander project moeten de project erg op elkaar lijken - d.w.z. het project waar u de voortgang naar toe kopieert (het bestemmingsproject) moet op een bepaald moment gecreëerd zijn als een kopie van het originele project (het bronproject). De voortgang wordt gekopieerd naar die taken en toewijzingen in het bestemmingsproject met interne ID's die matchen met de taken en toewijzingen in het bronproject. Wanneer u probeert om voortgang van het ene project naar een ander project dat er sterk van verschilt, zal de voortgangskopie niet slagen.

Om voortgang van het ene project naar een ander project te kopiëren:

  1. Open het project waarvan u de voortgang wilt kopiëren - het bronproject - en het project waarin u de voortgang wilt kopiëren - het bestemmingsproject.
  2. Klik op Overbrengen van Voortgang op het tabblad Start, in de Voortgang groep. Het Overbrengen Voortgang dialoogvenster verschijnt.
