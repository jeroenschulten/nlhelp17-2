Exporteren van resource inzet data

U kunt resource inzet data exporteren naar andere externe bestanden om te gebruiken in andere programma's. Dit zou handig kunnen zijn wanneer u rapporten moet aanleveren aan collega's die zelf geen gebruiken. De formaten waar u naartoe kunt exporteren zijn:

  • HTML.
  • CSV.

U kunt resource inzet data in het komma-gescheiden variabelen formaat ook naar het Klembord exporteren, van waaruit u het dan kunt plakken in andere applicaties.

Voordat u de resource inzet data exporteert, zou u het zinvol kunnen vinden om de kolommen, koppen en cellen van de resource inzet view op te maken om te zorgen dat de data weergegeven wordt in de benodigde eenheden.

Exporteren van resource inzet data naar HTML

HTML is het formaat dat gebruikt wordt voor het publiceren van documenten op het internet, intranetten en extranetten. Bijvoorbeeld, Wanneer u maandelijks resource inzet rapporten op uw bedrijfsintranet wilt publiceren, kunt u resource inzet data exporteren naar HTML en vervolgens het HTML rapport linken met het intranet.

Wanneer u naar HTML exporteert wordt de resource inzet informatie naar een HTML tabel geconverteerd, met een layout die vergelijkbaar is met die van de resource inzet view zelf - d.w.z. met de groepeer rijen apart van de toewijzing rijen.

Om resource inzet data naar HTML te exporteren:

  1. Rechtsklik op een toewijzingsrij in de resource inzet view en selecteer Exporteer in het menu dat verschijnt. Het Resource Inzet Exporteren dialoogvenster verschijnt.
  2. Schakel het Uitvoeren naar bestand (HTML of CSV) selectievakje in.
  3. Voer het pad en de bestandsnaam (met een .HTM of .HTML extensie) in het Export bestandsnaam veld in om toe te passen op het bestand dat wordt aangemaakt wanneer u de resource inzet data exporteert.
  4. Klik op OK om het HTML bestand te creëren.
 Exporteren van resource inzet data naar CSV

Comma Separated Variable (CSV) bestanden kunnen worden gebruikt in applicaties zoals spreadsheets.

Wanneer u naar een CSV bestand exporteert bevat de uitvoer de toewijzing rijen met de groeperingswaarden herhaald voor elke rij, hetgeen het makkelijker maakt om de data in een spreadsheet te manipuleren.

Om resource inzet data naar een CSV bestand te exporteren:

  1. Rechtsklik op een toewijzing rij in de resource inzet view en selecteer Exporteer in het menu dat verschijnt. Het Resource Inzet Exporteren dialoogvenster verschijnt.
  2. Schakel het Uitvoeren naar bestand (HTML of CSV) selectievakje in.
  3. Voer het pad en de bestandsnaam (met een .CSV extensie) in het Export bestandsnaam veld in om toe te passen op het bestand dat wordt aangemaakt wanneer u de resource inzet data exporteert.
  4. Klik op OK om het CSV bestand te creëren.
 Kopiëren van resource inzet data naar het Klembord

Om resource inzet data naar het Klembord te kopiëren:

  1. Rechtsklik op een toewijzing rij in de resource inzet view en selecteer Exporteer in het menu dat verschijnt. Het Resource Inzet Exporteren dialoogvenster verschijnt.
  2. Schakel het Klembord kopie (CSV) selectievakje in.
  3. Klik op OK.

U kunt nu de informatie van het Klembord in een andere applicatie plakken.
