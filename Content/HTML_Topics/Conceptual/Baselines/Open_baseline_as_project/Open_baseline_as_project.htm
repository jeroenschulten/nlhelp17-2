Een baseline als een afzonderlijk project openen

U kunt een baseline openen als een afzonderlijk project door, op dezelfde wijze als dat u een project opent. Het openen van een baseline als een afzonderlijk project is de enige wijze waarop u notities kunt zien die opgeslagen zijn binnen de baseline.

Om een baseline te openen als een afzonderlijk project:

Wanneer u een baseline als project opent, geeft de titel van het tabblad de baseline naam en bestandsnaam weer. Als u een embedded baseline als project opent, is de bestandsnaam dezelfde als die van het project zelf, en daarom wordt de titel van het tabblad dan voorafgegaan door de tekst "embedded baseline -" om het te onderscheiden van het project:

U kunt een baseline ook openen als een afzonderlijk project vanuit het Bestand tabblad van de Backstage view, zoals hieronder beschreven.

Openen van een baseline van een lokaal project als een afzonderlijk project

Om een baseline van een lokaal project als een afzonderlijk project te openen:

  1. Wanneer u aan het werken bent met een baseline die in een lokaal project embedded is, exporteer dan de baseline naar een afzonderlijk bestand.

    U moet dit doen met embedded baselines voordat u ze kunt openen als een afzonderlijk project vanuit het Bestand tabblad van de Backstage view; u hoeft dit niet te doen vóór het openen van een baseline als een afzonderlijk project vanuit het Baseline/What If Manager dialoogvenster.
  2. Klik op het Bestand tabblad, en klik vervolgens op Openen.
  3. Klik op Bladeren. Het Selecteren dialoogvenster verschijnt.
  4. Selecteer Baseline Bestanden (*.ppb; *.mdb) in het Bestandstype veld.
  5. Klik op Lokale Projecten en navigeer naar de map waarin de baseline staat opgeslagen.
  6. Selecteer de baseline en klik op Openen.
 Openen van een baseline van een Enterprise project als een afzonderlijk project

Om een baseline van een Enterprise project als afzonderlijk project te openen:

  1. Klik op het Bestand tabblad, en klik vervolgens op Openen.
  2. Klik op Enterprise. Een navigeerbare lijst met datasources en projecten verschijnt.
  3. Selecteer Baselines in het veld dat verschillende typen project weergeeft.
  4. Navigeer door de datasources en projecten om de baseline te lokaliseren die u wilt openen.
  5. Klik op the baseline om deze te openen.

De volgende regels zijn van toepassing wanneer u een baseline als een afzonderlijk project opent, behalve wanneer u een baseline hebt geopend die naar een afzonderlijk project is geëxporteerd:

  • U kunt geen baseline maken van een baseline die geopend is als een afzonderlijk project
  • Wanneer een baseline geopend is als een afzonderlijk project, kunt u die niet weergeven in het originele project.
  • Wanneer een baseline geopend is als een afzonderlijk project, kunt u die niet verwijderen
