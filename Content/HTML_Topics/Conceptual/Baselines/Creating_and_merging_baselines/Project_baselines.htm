Verschillende baselines associëren aan individuele projecten

Wanneer u een portfolio van projecten heeft opgebouwd binnen één bestand door afzonderlijke kaarten en samenvattingsgroepen als project te identificeren, kunt u een andere baseline aan elk individueel project associëren. Door dit te doen bent u in staat om nieuwe en bijgewerkte gegevens afzonderlijk samen te voegen en kunt u baseline informatie tonen in de tabel en op de balkkaart die alleen te maken heeft met het op dat moment zichtbare project. Nadat een baseline geassocieerd is aan een project, staat het bekend als een 'project baseline'.

Om een baseline te associëren aan een individueel project:

  1. Creëer een nieuwe baseline die alleen informatie bevat met betrekking tot de kaart of samenvattingsgroep die als project gepresenteerd wordt. Doe dit door te klikken op de kaart of samenvattingsgroep in de project view om de taken te zien binnen het afzonderlijke project en te kiezen om een baseline te creëren waarin alle taken in de huidige view zitten. Wanneer project baselines gecreëerd worden heeft het geen zin om andere informatie - bijv. gegevens uit andere projecten - in de baseline op te nemen.
  2. Klik met de rechtermuisknop op het kaart of samenvattingsgroep pictogram in de project view dat een project representeert en selecteer Eigenschappen. Het Kaart/Project Eigenschappen dialoogvenster verschijnt.
  3. Klik op het tabblad Project.
  4. Selecteer de baseline die de informatie over het project bevat in het Project Baseline veld.

Wanneer u verschillende baselines aan individuele projecten binnen een portfolio gekoppeld heeft, kunt u informatie van de project baselines op de balkkaart tonen door het inschakelen van de Project Baseline selectievakjes op het tabblad Baselines van het Opmaak Balkkaart dialoogvenster tabel velden configureren om informatie van de project baselines te tonen door Project Baseline te selecteren in het Gegevens veld op het tabblad Kolom Definities van het Tabel Definitie Eigenschappen dialoogvenster. U kunt een aantal project baseline-gerelateerde velden in de tabel tonen.
