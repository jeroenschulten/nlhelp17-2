Importeren van een project als een baseline

Misschien wilt u een project importeren als een baseline. Voorbeeld: u boekt een deel van het project uit, bewerkt het en importeert het weer in het originele project als baseline. Of, u wilt een baseline importeren in het originele project, bijvoorbeeld als u per ongeluk een baseline verwijdert, en u een back-up nodig hebt. Wanneer u een project geïmporteerd hebt als een baseline, wordt deze automatisch in het project embedded.

Wanneer u een project of baseline importeert, kunt u

U kunt ook bestanden importeren die zijn opgeslagen in het Primavera XER-formaat als baselines. Als u een XER-bestand als baseline importeert, geeft u niet op hoe taken in het XER-bestand moeten worden gemapt aan taken in het actieve project; taken worden in deze situatie altijd gemapt met behulp van de unieke taak-ID's.

U kunt geen baselines importeren die al in het project worden weergegeven en u kunt een project niet in zichzelf importeren. Als u een project (bijvoorbeeld een uitgeboekt project) weer in zichzelf wilt importeren, hernoemt u het project voordat u deze importeert. Het importeren van een project of baseline kan niet ongedaan worden gemaakt, maar u kunt een geïmporteerde baseline op dezelfde manier verwijderen als bij het verwijderen van andere baselines.

Om een project of baseline te importeren:

  3. Kies het project of de baseline die u wilt importeren, en klik vervolgens op Open. Het Baseline Import Opties dialoogvenster verschijnt.
  4. Schakel het Embed baseline selectievakje in om het project of de baseline die u aan het importeren bent in het actieve project bestand te embedden, of schakel het selectievakje uit om het project of de baseline te importeren als een apart bestand, met een .PPB bestandsextensie.
  5. Wanneer u voor embedden kies, selecteer het juiste keuzerondje om de manier op te geven waarop taken in het project of de baseline die u importeert worden gemapt met de taken in het actieve project.
  6. Klik op OK. Het project of de baseline wordt geïmporteerd.

Importeren van een XER bestand als een baseline

Om een XER bestand als een baseline te importeren:

  3. Kies het XER bestand dat u wilt importeren, en klik vervolgens op Openen. Het Primavera XML & XER Import Opties dialoogvenster verschijnt.
  4. Specificeer welke kalender is toegepast op de link-wachttijden in het project dat u aan het importeren bent in .
  5. Klik op OK. Het XER bestand wordt nu geïmporteerd als een baseline.

    Wanneer

Wanneer het importeren voltooid is, wordt het geïmporteerde project, baseline of XER bestand toegevoegd aan de lijst van baselines in de Baseline/What If Manager. U gebruikt het tabblad Baselines in het dialoogvenster Opmaak Balkkaart om de nieuwe baseline te laten weergeven.
