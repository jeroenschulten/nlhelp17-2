Importeren van uitzonderingen in kalenders vanuit Microsoft Excel

U kunt identieke uitzonderingen in een kalender creëren van een serie datums, door een Microsoft Excel bestand te importeren die een lijst met datums bevat.

Voorbeelden van Microsoft Excel bestanden waarvan u uitzonderingen kunt importeren

Het Microsoft Excel bestand kan op elke mogelijke manier zijn opgemaakt, mits het maar één of meer datums bevat.

Om uitzonderingen te importeren in een kalender vanuit een Microsoft Excel bestand:

  1. Rechtsklik in de project view op de kalender waarin u de uitzonderingen wilt importeren en selecteer Importeer Uitzonderingen. Het Importeer Uitzonderingen in Kalender dialoogvenster verschijnt.
