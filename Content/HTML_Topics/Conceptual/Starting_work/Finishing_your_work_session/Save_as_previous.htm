Het project opslaan in een vorig bestandsformaat

Het is mogelijk om projecten beschikbaar te maken voor gebruikers die een oudere versie van hebben door middel van het opslaan in een vorig bestandsformaat.

Om een project in een vorig bestandsformaat op te slaan:

  1. Klik in het tabblad Bestand, klik vervolgens op Opslaan Als.
  2. Klik op Deze PC om aan te geven dat u het project op een locatie op uw computer of op een gemapt station moet worden opgeslagen.
  3. Navigeer naar de map waarin het project opgeslagen dient te worden.
  4. Voer een geschikte bestandsnaam voor het project in.
  5. Selecteer de versie waarvan u bestandstype wilt gebruiken in het veld onder de bestandsnaam.
  6. Klik Opslaan. Het project is opgeslagen in een vorig bestandsformaat van .

Merk op dat wanneer een project in een vorig bestandsformaat opgeslagen is, het originele project in geopend blijft - niet de versie van het project dat in een vorig bestandsformaat is opgeslagen.

Let er verder ook op dat Baselines die bij het originele project horen niet opgeslagen worden naast de versie van het project die opgeslagen is in een vorig bestandsformaat.

Versie 14 van introduceerde een verbeterde kleurselector, waardoor het mogelijk werd om uit een breder scala aan kleuren te kiezen dan voorheen beschikbaar was. Als u een project opslaat waarin een kleur die niet eerder beschikbaar was, in een vorige bestandsindeling wordt gebruikt, wordt de kleur gewijzigd in de dichtstbijzijnde beschikbare kleur.

Het is ook mogelijk om gegevens van uit te boeken in een vorig bestandsformaat.
