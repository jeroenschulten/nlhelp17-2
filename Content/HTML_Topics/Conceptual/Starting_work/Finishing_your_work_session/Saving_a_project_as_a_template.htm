Een project opslaan als een sjabloon

Wanneer u in een project ingelogd bent als de hoogst-niveau gebruiker kunt u U kunt het project opslaan als een projectsjabloon in de map die ingesteld staat bij het Sjabloon-project pad op het tabblad Bestandslocaties van het dialoogvenster Opties. Wanneer en project is opgeslagen als een sjabloon, dan kunt u nieuwe projecten aanmaken die gebaseerd zijn op het sjabloon. Wanneer en nieuw project is aangemaakt, dan bevat het dezelfde attributen als het sjabloonbestand. Dit is een heel handige manier om meerder projecten aan te maken die allemaal dezelfde attributen (kalenders, resources, codes enz) bevatten.

Een project opslaan als een sjabloon:

  1. Klik in het tabblad Bestand, klik vervolgens op Opslaan Als.
  2. Klik op Deze PC om aan te geven dat u het project op een locatie op uw computer of op een gemapt station moet worden opgeslagen.
  3. Navigeer naar de map die is aangegeven in het Sjabloon-project pad op het tabblad Bestandslocaties in het dialoogvenster Opties in het veld Opslaan in.
  4. Voer een geschikte bestandsnaam voor het sjabloon in.
  5. Klik op Opslaan om het project als sjabloon op te slaan.
