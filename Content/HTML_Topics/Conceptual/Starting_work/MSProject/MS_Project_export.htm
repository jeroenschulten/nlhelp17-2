Projecten exporteren naar Microsoft Project

projecten kunnen naar XML bestanden worden geëxporteerd zodat ze geopend kunnen worden in Microsoft Project. U kunt projecten ook rechtstreeks exporteren naar Microsoft Project's MPP bestandsformaat, mits u Microsoft Project geïnstalleerd heeft op dezelfde computer als .

Behalve dat u in staat bent om gehele projecten naar Microsoft Project te exporteren, wanneer Microsoft Project geïnstalleerd heeft op dezelfde computer, kunt u ervoor kiezen om alleen de huidige groep of view naar het XML of het MPP formaat te exporteren.

Let op, vanwege beperkingen van de XML en MPP bestandsformaten, kan informatie van uw projecten verloren gaan aangezien veel van de gegevens niet naar XML of MPP formaat geëxporteerd kunnen worden.

Om een project naar Microsoft Project MPP formaat te exporteren:
  1. Om het gehele project naar MPP te exporteren, klik Klik op het tabblad Bestand, en dan op Import/Export, en klik vervolgens op Exporteer naar Microsoft Project; Om alleen de huidige groep of view te exporteren, rechtsklik in de project view op een kaart of samenvattingsgroep en selecteer Exporteer groep of view naar Microsoft Project. Het Export dialoogvenster verschijnt. Merk op dat u meer dan één groep tegelijk kunt exporteren door het CTRL-selecteren van de groepen voordat u rechtsklikt.
  3. Voer indien gewenst een nieuwe bestandsnaam in voor het project.
  4. Selecteer Microsoft Project (*.mpp) in het Opslaan als type veld.
  6. De Export domein pagina van de wizard verschijnt.
  8. De Objecten pagina van de wizard verschijnt.
  16. Wanneer u ervoor gekozen heeft om codebibliotheken te exporteren, gebruikt u de codebibliotheken pagina van de wizard om te specificeren welke codebibliotheken u wilt exporteren.
  17. Het project is nu opgeslagen als een MPP bestand dat geopend kan worden in Microsoft Project.
Om een project naar Microsoft Project XML formaat te exporteren:
  1. Om het gehele project naar XML te exporteren, klik Klik op het tabblad Bestand, en dan op Import/Export, en klik vervolgens op Exporteer naar Microsoft Project; Om alleen de huidige groep of view te exporteren, rechtsklik in de project view op een kaart of samenvattingsgroep en selecteer Exporteer groep of view naar Microsoft Project. Het Export dialoogvenster verschijnt. Merk op dat u meer dan één groep tegelijk kunt exporteren door het CTRL-selecteren van de groepen voordat u rechtsklikt.
  3. Voer indien gewenst een nieuwe bestandsnaam in voor het project.
  4. Selecteer Microsoft Project XML formaat (*.xml) in het Opslaan als type veld.
  6. De Export domein pagina van de wizard verschijnt.
  8. De Objecten pagina van de wizard verschijnt.
  16. Wanneer u ervoor gekozen heeft om codebibliotheken te exporteren, gebruikt u de codebibliotheken pagina van de wizard om te specificeren welke codebibliotheken u wilt exporteren.
  17. Het project is nu opgeslagen als een XML bestand dat geopend kan worden in Microsoft Project.
