Importeren van projecten van Microsoft Project

U kunt Microsoft® Project projecten die zijn opgeslagen als XML bestanden importeren in . U kunt ook Microsoft Project MPP bestanden in importeren, op voorwaarde dat u Microsoft Project op dezelfde computer heeft geïnstalleerd als .

Behalve dat u in staat bent om Microsoft Project projecten als complete eigen nieuwe projecten te importeren, wanneer u Microsoft Project op dezelfde computer heeft geïnstalleerd, kunt u er ook voor kiezen om Microsoft Project projecten die opgeslagen zijn als MPP of XML bestanden in specifieke kaarten of samenvattingsgroepen binnen bestaande projecten te importeren.

Let op dat vanwege verschillen in bestandsformaat, bepaalde informatie van XML en MPP bestanden kan veranderen of verloren gaan bij het importeren naar .

Om een Microsoft Project MPP of XML bestand in te importeren:

  1. Om een bestand als een zelfstandig project te importeren, klik Klik op het tabblad Bestand, en dan op Import/Export, en klik vervolgens op Importeer van Microsoft Project; om een bestand in een specifieke kaart of samenvattingsgroep te importeren, rechtsklik op een kaart of samenvattingsgroep in de project view en selecteer Importeer in kaart vanuit Microsoft Project. Het Microsoft Project Import Opties dialoogvenster verschijnt. Dit dialoogvenster verschilt afhankelijk of u een zelfstandig project als project in aan het importeren bent of dat u een bestand in een specifieke kaart of samenvattingsgroep aan het importeren bent.
  3. Wanneer u een bestand als een zelfstandig project aan het importeren bent, selecteer Selecteer een sjabloon in het Gebruik sjabloon veld om een sjabloon toe te passen op het project, of kies Blanco in het veld om het project te importeren zonder sjabloon.
  4. Wanneer u een bestand in een specifieke kaart of samenvattingsgroep aan het importeren bent, schakel dan het Plaats geïmporteerde bibliotheek objecten in een benoemde map selectievakje in om
  5. Schakel het Importeer kostentoewijzingen selectievakje in
  6. Schakel het Importeer resourcetoewijzingen selectievakje in
  7. Schakel het Import codebibliotheken selectievakje in
  8. Gebruik het Converteer negatieve kosten in veld om te specificeren
  9. Klik op OK om het geselecteerde Microsoft Project bestand te importeren.

Merk op dat wanneer u een Microsoft Project bestand in importeert dat haar programma-kaart wordt weergegeven als een project..

Nadat u een Microsoft Project bestand in geïmporteerd heeft, wordt u geadviseerd om het project op te slaan als 's bestandsformaat - als PP bestand - om te voorkomen dat informatie die niet door de XML en MPP formaten ondersteund wordt verloren gaat.
