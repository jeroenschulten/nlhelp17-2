Codes toewijzen aan balken, taken en toewijzingen

Wijs een code toe aan een gehele balk, taak of toewijzing door één van de volgende mogelijkheden te kiezen:

  • Selecteer de balk, taak of toewijzing en selecteer vervolgens een code uit het vak op de werkbalk Object bewerken. Klik met de rechtermuisknop op dit vak om een andere codebibliotheek te selecteren.
  • Sleep de code uit Bibliotheek verkenner of de project view en plaats deze op de balk, taak of toewijzing.
  • Gebruik de Codes sectie van het tabblad Balk of Taak in het dialoogvenster Balk en Taak Eigenschappen of op het bijbehorende dialoogvenster Toewijzing Eigenschappen.

In aanvulling op de technieken zoals hierboven omschreven, kunt u een code aan een taak of toewijzing toewijzen door met de rechter muisknop te klikken op de taak of toewijzing, en vervolgens te kiezen voor Alloceer Code en een code te selecteren uit de hiërarchische codebibliotheken.

Om dezelfde code toe te wijzen aan al de taken op een balk, selecteer de balk, en wijs de code aan één van taken toe op de balkregel.

Afhankelijk van de manier waarop elke codebibliotheek is opgezet, kunt u wellicht meer dan één code toekennen van een enkele codebibliotheek aan een object. Om een tweede of volgende code van een codebibliotheek toe te wijzen aan een object door middel van slepen, drukt u CTRL in en sleept u de code vanuit de project view of de bibliotheek verkenner naar het object

Als u codes hebt toegewezen aan balken en taken, kunt u filteren en sorteren/groeperen op balken of taken met bepaalde codes. Bijvoorbeeld, u kunt een view maken van alleen taken met een code voor een hoog risico factor.

U kunt kiezen om de opmaak van elk codebibliotheek item naast de naam weer te geven, wanneer deze verschijnt in de project view en in de Object Bewerken werkbalk. Het weergeven van de opmaak van codebibliotheekitems in de project view vergemakkelijkt het voor u om te controleren dat u de juiste code aan de objecten in de balkkaart.
