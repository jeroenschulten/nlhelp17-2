Een code breakdown structuur weergeven

U kunt nu alle balken en taken tonen waaraan één of meer codes zijn toegewezen door een code breakdown structure te tonen. Een code breakdown structure toont alle balken en taken waaraan de geselecteerde codes zijn toegewezen, met een subtitel die een "nep" samenvattingstaak bevat, die de vroegste en laatste datums van de items aangeeft, weergegeven voor elke code:

Om een code breakdown structuur weer te geven die alle balken en taken waaraan de geselecteerde codes zijn toegewezen, rechtsklikt u op een code-bibliotheek of code-bibliotheek item in de project view en selecteert u Open Code Breakdown Structure: De code breakdown verschijnt zoals hieronder geïllustreerd wordt, met een subtitel die de geselecteerde code weergeeft en de balken en taken waaraan de code toegewezen is onder de subtitel:

Merk op dat de subtitel die de geselecteerde code weergeeft niet verschijnt wanneer u een code breakdown structuur in een view weergeeft waarop een sortering/groepering is toegepast.

Om een code breakdown structuur van alle balken en taken waaraan de codes binnen een codebibliotheek zijn toegewezen, rechtsklikt u op een codebibliotheek map in de project view en selecteert u Open Code Breakdown Structure.

Nadat een code breakdown structuur weergegeven is, kunt u de code veranderen waarnaar ze verwijst door op een andere code in de project view te klikken. Telkens wanneer u op een andere code klikt, wordt de view bijgewerkt om alle balken en taken waaraan de andere code is toegewezen te tonen. U kunt ook de view bijwerken door er meer dan een individuele code aan toe te voegen. Om dit te doen, CTRL-klikt u op de codes in de project view. De code breakdown structure wordt dan bijgewerkt op ook de balken en taken waaraan de erbij geselecteerde codes zijn toegewezen te tonen:

Wanneer een taak toewijzingen heeft van meer dan een van de geselecteerde codes heeft, wordt deze in de lijst getoond in de code breakdown structuur onder elke eraan toegewezen code.

Om een code breakdown structuur te sluiten, klik op een van de kaarten of samenvattingsgroepen in de projecthiërarchie.

Specificeren van de kleuren waarmee code breakdown structuur subtitels worden weergegeven

Om de kleuren te specificeren waarmee code breakdown structuur subtitels worden weergegeven:

  1. Klik op het Bestand tabblad, en klik vervolgens op Opties. Het Opties dialoogvenster verschijnt.
  2. Klik op het Tabel tabblad.
  3. Schakel het Gebruik vaste bandkleur
  4. in het Voorgrond veld.
  5. in het Achtergrond veld.
