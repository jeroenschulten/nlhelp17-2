Aanpassen van de Object Bewerken werkbalk

U kunt specificeren welke opdrachten opgenomen moeten worden op de Object Bewerken werkbalk.

U kunt opdrachten toevoegen aan de Object Bewerken werkbalk, opdrachten verwijderen uit de werkbalk, en de locatie wijzigen van de opdrachten op de werkbalk. Wanneer u een macro-commando aan de Object Bewerken werkbalk heeft toegevoegd kunt u de naam en het icoontje ervan wijzigen.

Om een opdracht toe te voegen aan de Object Bewerken werkbalk
  1. Rechtsklik op het Lint en selecteer Aanpassen. Het tabblad Lint van het Lint Aanpassingen dialoogvenster verschijnt.
  2. Klik op het Object Bewerken werkbalk tabblad. Het grote veld aan de linkerkant - de Opdrachtenlijst - toont de verschillende opdrachten die aan de Object Bewerken werkbalk kunnen worden toegevoegd, terwijl het grote veld aan de rechterkant - de Object Bewerken werkbalk Opdrachtenlijst - een lijst van de opdrachten toont die op dit moment opgenomen op de werkbalk.
  3. Standaard toont de Opdrachtenlijst alle beschikbare opdrachten; U kunt de lijst opdrachten die in deze lijst staan verkleinen door een categorie te selecteren in het Kies Opdrachten van veld. Om dit te doen, selecteer:
    • Alle Opdrachten om alle beschikbare opdrachten te tonen, ongeacht of ze wel of niet worden getoond op het Lint.
    • Opdrachten niet in het Lint om juist alle opdrachten te tonen die op dit moment niet op het Lint staan.
    • Object Bewerken om alle opdrachten te tonen die weer te geven zijn op de Object Bewerken werkbalk.
    • Macro's om uw gebruikersmacro's te tonen.
    • Een specifiek Lint tabblad om alle Opdrachten te tonen die op dat specifieke tabblad staan.
  4. Select de opdracht die u wilt toevoegen aan de Object Bewerken werkbalk in de Opdrachtenlijst.
  5. Klik op Toevoegen. De geselecteerde opdracht wordt toegevoegd aan het einde van de Object Bewerken werkbalk Opdrachtenlijst.
Om de positie van een opdracht op de Object Bewerken werkbalk te veranderen
  1. Rechtsklik op het Lint en selecteer Aanpassen. Het tabblad Lint van het Lint Aanpassingen dialoogvenster verschijnt.
  2. Klik op het tabblad Object Bewerken werkbalk.
  3. Lokaliseer de opdracht die u wilt verplaatsen in het grote veld aan de rechterkant - de Object Bewerken werkbalk Opdrachtenlijst.
  4. Klik op om de geselecteerde opdracht één plaats naar boven te verplaatsen in de lijst - waardoor ze een plaats naar links zal verplaatsen op de werkbalk - of klik op één plaats naar beneden te verplaatsen - waardoor ze een plaats naar rechts zal verplaatsen op de werkbalk.
 Om een opdracht van de Object Bewerken werkbalk te verwijderen
  1. Rechtsklik op het Lint en selecteer Aanpassen. Het tabblad Lint van het Lint Aanpassingen dialoogvenster verschijnt.
  2. Klik op het tabblad Object Bewerken werkbalk.
  3. Lokaliseer de opdracht die u wilt verwijderen in het grote veld aan de rechterkant - de Object Bewerken werkbalk Opdrachtenlijst.
  4. Klik op Verwijderen.
 Om de naam, omschrijving en icoontje van een macro-commando die u heeft toegevoegd aan de Object Bewerken werkbalk te wijzigen
  1. Rechtsklik op het Lint en selecteer Aanpassen. Het tabblad Lint van het Lint Aanpassingen dialoogvenster verschijnt.
  2. Klik op het tabblad Object Bewerken werkbalk.
  3. Lokaliseer het macro-commando dat u wilt wijzigen in het grote veld aan de rechterkant - de Object Bewerken werkbalk Opdrachtenlijst.
  4. Klik op Wijzigen. Een dialoogvenster verschijnt waarin u de naam, omschrijving en icoontje van het macro-commando kunt wijzigen.
  5. Voer een nieuwe naam en omschrijving in voor het macro-commando in het Weergave naam veld, selecteer desgewenst een ander icoontje en klik op OK.
