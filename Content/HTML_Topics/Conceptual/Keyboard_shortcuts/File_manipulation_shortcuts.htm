Om Druk op
Een nieuw project starten CTRL N
Een bestaand project openen CTRL O
De actieve view afdrukken CTRL P
Het actieve project opslaan CTRL S
Het actieve project opslaan onder een andere naam F12
Het actieve venster sluiten CTRL F4
afsluiten ALT F4
