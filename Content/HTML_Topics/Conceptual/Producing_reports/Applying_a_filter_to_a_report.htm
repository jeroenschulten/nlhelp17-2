Een filter toepassen op een resourcesrapport

U gebruikt het tabblad Filter in het dialoogvenster Resource Rapport om een filter toe te passen op een resourcesrapport. Wanneer u een filter toepast op een resourcesrapport, bevat het rapport alleen informatie die past binnen de filtercriteria.

Wanneer u een filter wilt toepassen op een resourcesrapport, selecteert u het filter dat u wilt toepassen.

U kunt de eigenschappen van een filter bewerken, door te klikken op Eigenschappen.
