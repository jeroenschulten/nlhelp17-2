Code/linkcategorie

Wanneer er op dit moment een taak in de balkaart geselecteerd is, geeft het besturingselement de codes weer waaraan de geselecteerde taak is toegewezen. Om een andere code aan de geselecteerde taak of link toe te wijzen, klikt u op dit besturingselement en selecteert u een code uit de keuzelijst die verschijnt. Het selecteren van een andere codebibliotheek in dit besturingselement heeft geen invloed op welke codebibliotheken op dit moment worden weergegeven op de balkkaart.

Wanneer er op dit moment een link in de balkaart geselecteerd is, geeft het besturingselement de linkcategorie weer waaraan de geselecteerde link is toegewezen.

Om een andere code of linkcategorie aan de geselecteerde taak of link toe te wijzen, klik op het besturingselement en selecteer een code of categorie uit de lijst die verschijnt.

Wanneer u het Toon codebibliotheek opmaak selectievakje op het tabblad View van het Opties dialoogvenster ingeschakeld heeft, zal de opmaak van elke code in dit besturingselement verschijnen naast de naam ervan. Dit vergemakkelijkt het controleren dat u de juiste code aan de objecten in de balkkaart heeft toegewezen.
